Introduction

Description : La tache MAILOToIMGIReader est utilisée pour lire et traiter les fichiers MAILO, en particulier pour renommer ces fichiers selon un format spécifique basé sur des règles de nommage prédéfinies.

Package

Emplacement : sas.action.readers.ampup

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers MAILO dans le répertoire configuré.

Traitement :

Lit le fichier pour extraire les informations nécessaires à partir de son nom.

Détermine le code de la banque destinataire et génère un nouveau nom de fichier basé sur ce code et d'autres paramètres.

Renomme le fichier selon le format requis.

Résultat : Le fichier est renommé et prêt à être utilisé dans les processus suivants.

En cas d’erreur : Les erreurs de renommage sont consignées dans les journaux et une notification d'erreur est affichée si possible.

Opérations Clés

Lecture des Fichiers : Analyse le nom du fichier pour extraire les informations nécessaires.

Génération de Noms : Utilise des méthodes utilitaires pour générer un nouveau nom de fichier basé sur le code de la banque destinataire et d'autres paramètres.

Renommage des Fichiers : Renomme le fichier avec le nouveau nom généré. Gère les erreurs de renommage en affichant des notifications et en lançant des exceptions.

Paramètres du Composant

CODE_BANQUE_SICA3 : Code de la banque utilisé pour déterminer le type de fichier à générer, avec comme valeur par défaut: CI160.

DATECOMPENS_NAT et DATECOMPENS_SRG : Dates utilisées pour calculer les séquences de fichiers nationaux et régionaux respectivement.

Cette fiche technique inclut les fonctionnalités principales de la classe MAILOToIMGIReader, en mettant l'accent sur le processus de renommage des fichiers et la gestion des erreurs associées.
