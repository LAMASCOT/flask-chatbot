Introduction

Description : La tache RejetNonComChequeCorporatesWriter est une extension de FlatFileWriter utilisée pour générer des fichiers de rejet pour les chèques non compensables des entreprises. Elle traite les chèques non compensables rejetés, génère un fichier plat contenant les détails des rejets, et enregistre ces fichiers dans un répertoire spécifié.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Générer des fichiers de rejet pour les chèques non compensables des entreprises, en traitant les chèques rejetés et en générant un fichier plat avec les détails des rejets.

Traitement :

Récupère les chèques non compensables rejetés depuis la base de données.

Génère un fichier plat contenant les détails des chèques rejetés pour chaque établissement.

Enregistre les fichiers générés dans le répertoire spécifié par SIB_IN_FOLDER.

Méthodes

execute() : Exécute le processus de génération des fichiers de rejet, en récupérant les chèques non compensables rejetés, en générant les fichiers plats, et en enregistrant les événements dans les logs.

Paramètres Utilisés

CETAOPENONCOMREJ : Paramètre utilisé pour identifier l'état des chèques non compensables rejetés.

CETAOPEANO : Paramètre utilisé pour identifier un autre état des chèques non compensables.

SIB_IN_FOLDER : Répertoire de destination pour les fichiers de rejet générés.

Remarques

Gestion des Erreurs : Les erreurs lors du traitement des chèques et de la génération des fichiers sont enregistrées dans les logs.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de vérifications ou de formats de fichiers si nécessaire.

Configuration : La classe utilise des composants configurables pour le traitement des chèques et la génération des fichiers, permettant une adaptation facile aux besoins spécifiques du système.

Fichier Plat : Le fichier plat généré contient les détails des chèques rejetés, y compris l'origine, le numéro de chèque, et le motif de rejet, et est enregistré dans le répertoire spécifié par SIB_IN_FOLDER.
