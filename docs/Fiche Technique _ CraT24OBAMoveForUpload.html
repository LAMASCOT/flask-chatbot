Introduction

Description : La tache CraT24OBAMoveForUpload est une extension de FlatFileWriter utilisée pour déplacer des fichiers CRA (Comptes Rendus d'Activité) vers un répertoire de destination pour le système T24. Elle surveille un répertoire source pour des fichiers spécifiques et les déplace tout en gérant les sauvegardes des fichiers existants.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Déplacer les fichiers CRA vers un répertoire de destination pour le système T24.

Traitement :

Surveille un répertoire source pour des fichiers avec des extensions spécifiques.

Déplace les fichiers trouvés vers un répertoire de destination.

Gère les sauvegardes des fichiers existants avant de les écraser.

Méthodes

CraT24OBAMoveForUpload() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de déplacement des fichiers CRA vers le répertoire de destination.

moveAndOverwrite(File source, File dest) : Déplace un fichier source vers une destination, en sauvegardant le fichier de destination s'il existe déjà.

getNonExistingTempFile(File inputFile) : Génère un nom de fichier temporaire qui n'existe pas encore dans le même répertoire que le fichier d'entrée.

end() : Termine le processus en enregistrant un événement de log.

Paramètres Utilisés

SIB_IN_FOLDER : Répertoire source où se trouvent les fichiers CRA.

DEST_CRA_FOLDER : Répertoire de destination pour les fichiers CRA déplacés.

Remarques

Gestion des Erreurs : Les erreurs lors du déplacement des fichiers sont gérées par des logs, et la description de l'opération est mise à jour pour refléter le statut de chaque étape.

Extensibilité : La classe peut être étendue pour inclure d'autres types de fichiers ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons IO : La classe utilise FileUtils de la bibliothèque Apache Commons IO pour les opérations de copie de fichiers.
