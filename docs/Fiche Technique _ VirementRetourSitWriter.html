Introduction

Description : La tache VirementRetourSitWriter est une extension de FlatFileWriter utilisée pour envoyer des virements de retour vers le SIB (Système d'Information Bancaire). Elle génère des fichiers plats contenant les informations des virements et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer des virements de retour vers le SIB en générant des fichiers plats et mettre à jour leur état dans la base de données.

Traitement :

Vérifie si le verrouillage des virements de retour est actif.

Récupère les virements à envoyer en fonction de leur état.

Génère un fichier plat avec les informations des virements.

Met à jour l'état des virements après envoi.

Gère les messages associés aux virements reçus.

Méthodes

VirementRetourSitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des virements de retour vers le SIB.

getMessageVirementsRecus() : Retourne le message des virements reçus.

setMessageVirementsRecus(String messageVirementsRecus) : Définit le message des virements reçus.

Attributs

messageVirementsRecus : Message des virements reçus.

dataBaseJPA : Instance de DataBaseJPA pour interagir avec la base de données.

Paramètres Utilisés

LOCK_VIR_RETOUR : Paramètre de verrouillage pour éviter les conflits d'accès lors de la composition des fichiers de virements retour.

CETAOPERET : État des virements retour.

CETAOPERETENVSIB : État des virements après envoi vers le SIB.

SIB_IN_FOLDER : Dossier de destination pour les fichiers générés.

VIR_OUT_FILE_ROOTNAME : Nom de base pour les fichiers de virements retour.

SIB_FILE_EXTENSION : Extension des fichiers générés.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des virements de retour sont gérées par des logs et des messages d'avertissement.

Extensibilité : La tache peut être étendue pour inclure d'autres types de virements ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et effectuer les mises à jour nécessaires.
