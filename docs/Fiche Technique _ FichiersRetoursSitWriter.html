Introduction

Description : La tache FichiersRetoursSitWriter est une extension de FlatFileWriter utilisée pour orchestrer l'envoi des fichiers de retour vers le SIB (Système d'Information Bancaire). Elle exécute plusieurs processus d'écriture pour différents types de fichiers de retour et gère les messages associés à chaque type de fichier.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Coordonner l'envoi des fichiers de retour vers le SIB en exécutant plusieurs écrivains de fichiers de retour et en collectant les messages associés.

Traitement :

Exécute une série de classes d'écriture de fichiers de retour, chacune responsable d'un type spécifique de fichier de retour.

Collecte et stocke les messages associés à chaque type de fichier de retour.

Méthodes

FichiersRetoursSitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des fichiers de retour en appelant les méthodes execute() de plusieurs classes d'écriture de fichiers de retour et collecte les messages associés.

Classes d'Écriture de Fichiers de Retour Exécutées

ChequeRetourSitWriter : Gère l'envoi des chèques retour.

EffetRetourSitWriter : Gère l'envoi des effets retour.

VirementRetourSitWriter : Gère l'envoi des virements retour.

ChequeRejeteSitWriter : Gère l'envoi des chèques rejetés.

EffetAllerRejeteSitWriter : Gère l'envoi des effets aller rejetés.

PrelevementRetourSitWriter : Gère l'envoi des prélèvements retour.

Attributs

messageChequesRecus : Message des chèques reçus.

messageVirementsRecus : Message des virements reçus.

messageEffetsRecus : Message des effets reçus.

messagePrelevementsRecus : Message des prélèvements reçus.

messageChequesRejetesRecus : Message des chèques rejetés reçus.

messageEffetsRejetesRecus : Message des effets rejetés reçus.

messagePrelevementsRejetesRecus : Message des prélèvements rejetés reçus.

Paramètres Utilisés

LOCK_EFF_RETOUR : Paramètre de verrouillage pour éviter les conflits d'accès lors de la composition des fichiers d'effets retour.

CETAOPERETREC : État des effets retour reçus.

CETAOPERET : État des effets retour.

CETAOPERETENVSIB : État des effets après envoi vers le SIB.

CETAOPERETRECENVSIB : État des effets retour reçus après envoi vers le SIB.

SIB_IN_FOLDER : Dossier de destination pour les fichiers générés.

EFF_OUT_FILE_ROOTNAME : Nom de base pour les fichiers d'effets retour.

Remarques

Extensibilité : La tache peut être étendue pour inclure d'autres types de fichiers de retour ou pour modifier l'ordre d'exécution des écrivains de fichiers.

Intégration : Cette tache sert de point central pour l'exécution coordonnée des processus d'écriture de fichiers de retour, facilitant ainsi la gestion des envois vers le SIB.
