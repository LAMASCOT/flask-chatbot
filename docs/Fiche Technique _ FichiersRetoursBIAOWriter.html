Introduction

Description : La tache FichiersRetoursBIAOWriter est une extension de FlatFileWriter utilisée pour orchestrer l'envoi des fichiers de retour vers le SIB (Système d'Information Bancaire) et SIGN. Elle exécute plusieurs processus d'écriture pour différents types de fichiers de retour.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Coordonner l'envoi des fichiers de retour vers le SIB et SIGN en exécutant plusieurs écrivains de fichiers de retour.

Traitement :

Exécute une série de classes d'écriture de fichiers de retour, chacune responsable d'un type spécifique de fichier de retour.

Méthodes

FichiersRetoursBIAOWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des fichiers de retour en appelant les méthodes execute() de plusieurs classes d'écriture de fichiers de retour.

Classes d'Écriture de Fichiers de Retour Exécutées

ChequeRetourSignWriter : Gère l'envoi des chèques retour vers SIGN.

ChequeRetourEISSitWriter : Gère l'envoi des chèques retour vers EIS.

EffetRetourSitWriter : Gère l'envoi des effets retour vers le SIB.

VirementRetourSitWriter : Gère l'envoi des virements retour vers le SIB.

ChequeRejeteSitWriter : Gère l'envoi des chèques rejetés.

Remarques

Extensibilité : La tache peut être étendue pour inclure d'autres types de fichiers de retour ou pour modifier l'ordre d'exécution des écrivains de fichiers.

Intégration : Cette tache sert de point central pour l'exécution coordonnée des processus d'écriture de fichiers de retour, facilitant ainsi la gestion des envois vers le SIB et SIGN.
