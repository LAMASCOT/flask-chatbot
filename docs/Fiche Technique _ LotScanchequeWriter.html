Introduction

Description : La tache LotScanchequeWriter est une extension de FlatFileWriter utilisée pour envoyer des lots de chèques scannés vers le Système d'Information Bancaire (SIB). Elle traite les chèques en attente, génère un fichier plat contenant les détails des chèques, et copie les images associées.

Package

Emplacement : sas.action.writers.borne

Description Fonctionnelle

Objectif : Générer et envoyer un fichier contenant les informations des chèques scannés vers le SIB.

Traitement :

Met à jour l'état des chèques avec des montants nuls ou vides.

Récupère les chèques et remises en attente depuis la base de données.

Génère un fichier plat pour chaque lot de chèques contenant les informations nécessaires.

Écrit les informations des chèques dans le fichier généré.

Copie les images des chèques dans le répertoire approprié.

Attributs

db : DataBase - Instance pour l'accès à la base de données.

Méthodes

LotScanchequeWriter() : Constructeur par défaut qui initialise la description du processus.

execute() : Exécute le processus de génération et d'envoi des fichiers de chèques scannés vers le SIB.

Paramètres Utilisés

CETAOPEERR : État d'erreur pour les chèques avec des montants nuls ou vides.

CETAREMSAIIRIS : État des remises en attente de traitement.

CETAOPEALLICOM1 : État des chèques après traitement réussi.

CETAREMENVSIB : État des remises après envoi réussi.

SIB_IN_FOLDER : Répertoire de destination pour les fichiers générés.

CHQ_IN_FILE_ROOTNAME : Nom de base pour les fichiers de chèques.

SIB_FILE_EXTENSION : Extension des fichiers de chèques.

AGENCE_BORNE : Code de l'agence pour les bornes.

DATECOMPENS_NAT : Date de compensation nationale.

SCANCHEQUES_IMG_FOLDER : Répertoire pour les images des chèques.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers et de la copie des images sont gérées par des logs.

Extensibilité : La classe est conçue pour être extensible, permettant l'ajout de nouvelles fonctionnalités ou l'intégration avec d'autres systèmes de traitement de chèques.

Utilisation de la Base de Données : Utilise DataBase et JDBCXmlReader pour gérer les connexions et les opérations sur la base de données.
