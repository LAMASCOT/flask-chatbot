Introduction

Description : La tache LotDeltaReader est conçue pour lire et traiter les fichiers de lots d'opérations bancaires, en extrayant les informations nécessaires et en mettant à jour les enregistrements dans la base de données.

Package

Emplacement : sas.action.readers.corporates

Fonctionnalités

Déclenchement : Activée lors de la détection de nouveaux fichiers de lots dans le répertoire configuré.

Traitement :

Lit chaque ligne du fichier pour extraire les informations des opérations bancaires.

Crée des objets Operation à partir des lignes lues.

Met à jour les enregistrements de chèques, virements, effets, et prélèvements dans la base de données si les conditions sont remplies.

Résultat : Les enregistrements sont mis à jour dans la base de données.

Gestion des Erreurs : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise un BufferedReader pour lire chaque ligne du fichier et extraire les informations nécessaires.

Création d'Objets :

Utilise la méthode getChamp pour extraire les champs des lignes et créer des objets Operation.

Mise à Jour des Enregistrements :

Recherche les enregistrements existants dans la base de données à l'aide de DataBaseJPA.

Met à jour les attributs des enregistrements si les conditions sont remplies.

Met à jour l'état des enregistrements si nécessaire.

Paramètres du Composant

Séparateur de Ligne : Utilise des méthodes spécifiques pour diviser les lignes en attributs.

Interactions avec la Base de Données

Requêtes SQL : Utilise des requêtes JPQL pour récupérer et mettre à jour les enregistrements de chèques, virements, effets, et prélèvements.

Gestion des Transactions : Utilise DataBaseJPA pour gérer les transactions de base de données.

Paramètres Utilisés et Valeurs Possibles

`StaticValues` :

Utilisation : Contient des constantes pour identifier les types d'opérations (e.g., VIR_CLIENT, CHQ_SCAN).

`Utility` :

Utilisation : Fournit des méthodes utilitaires pour la conversion de dates et d'autres opérations courantes.

`CMPUtility` :

Utilisation : Fournit des méthodes spécifiques au système pour obtenir des informations comme la date courante.
