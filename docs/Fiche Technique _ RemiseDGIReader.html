Introduction

Description : La tache RemiseDGIReader est utilisée pour lire et traiter les fichiers de remises DGI, en extrayant les informations des ordres de paiement et en les enregistrant dans la base de données.

Package

Emplacement : sas.action.readers.bni

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de remises DGI dans le répertoire configuré.

Traitement :

Lit chaque ligne du fichier pour extraire les informations des ordres de paiement.

Vérifie la cohérence des montants totaux des chèques avec le montant global renseigné.

Enregistre les informations des ordres de paiement dans la base de données.

Résultat : Les ordres de paiement sont insérés dans la base de données, avec vérification de la cohérence des données.

En cas d’erreur : Les erreurs de cohérence des montants ou d'en-tête manquante sont consignées dans les journaux et le traitement du fichier est interrompu.

Opérations Clés

Lecture des Fichiers : Utilise un Scanner pour lire chaque ligne du fichier et extraire les informations nécessaires.

Vérification de Conformité : Vérifie que le montant total des chèques correspond au montant global renseigné dans le fichier.

Interaction avec la Base de Données : Enregistre chaque ordre de paiement dans la base de données en utilisant DataBaseJPA.

Gestion des Erreurs : Consigne les erreurs de cohérence des montants et d'en-tête manquante dans les journaux.

Paramètres du Composant

Formats de Date : Utilise SimpleDateFormat pour formater les dates d'émission et comptables des ordres de paiement.

Cette fiche technique inclut les fonctionnalités principales de la classe RemiseDGIReader, en mettant l'accent sur le processus de lecture, de vérification et d'enregistrement des ordres de paiement.
