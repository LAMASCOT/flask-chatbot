Introduction

Description : La tache ChequeRejeteBAMLSitWriter est une extension de FlatFileWriter utilisée pour envoyer les chèques rejetés vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques rejetés à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques rejetés vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Vérifie si le verrouillage pour l'envoi des chèques rejetés est activé.

Récupère les chèques rejetés à envoyer.

Génère un fichier plat avec les informations des chèques rejetés.

Met à jour l'état des chèques après envoi.

Méthodes

ChequeRejeteBAMLSitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des chèques rejetés vers le SIB.

getMessageChequesRejetesRecus() : Retourne le message contenant le nombre de chèques rejetés et le montant total.

setMessageChequesRejetesRecus(String message) : Définit le message contenant le nombre de chèques rejetés et le montant total.

Paramètres Utilisés

CETAOPEREJRET : État des chèques rejetés prêts à être envoyés.

CETAOPEREJRETENVSIB : État des chèques rejetés après envoi.

LOCK_CHK_RJT_ALLER : Paramètre de verrouillage pour éviter les envois simultanés.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques rejetés sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et effectuer les mises à jour nécessaires.
