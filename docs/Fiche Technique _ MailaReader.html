Introduction

Description : La tache MailaReader est conçue pour lire et traiter les fichiers de type MAILA, en mettant à jour les états des enregistrements de chèques et d'effets dans la base de données.

Package

Emplacement : sas.action.readers.corporates

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers MAILA dans le répertoire configuré.

Traitement :

Lit le fichier pour extraire le nom sans suffixe.

Met à jour les états des enregistrements de chèques et d'effets dans la base de données en fonction du nom de fichier et des états initiaux et finaux.

Résultat : Les états des enregistrements de chèques et d'effets sont mis à jour dans la base de données.

En cas d’erreur : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise la méthode setFile pour définir le fichier à traiter.

Mise à Jour des Enregistrements :

Utilise des requêtes SQL pour mettre à jour les états des enregistrements de chèques et d'effets dans la base de données.

Utilise DataBaseJPA pour exécuter les mises à jour.

Paramètres du Composant

Nom de Fichier : Utilise Utility.removeFileNameSuffixe pour extraire le nom de fichier sans suffixe.

Interactions avec la Base de Données

Requêtes SQL : Utilise des requêtes SQL pour mettre à jour les états des enregistrements de chèques et d'effets.

Gestion des Transactions : Utilise DataBaseJPA pour gérer les transactions de base de données.

Paramètres Utilisés et Valeurs Possibles

`CETAOPEALLICOM1ACC` :

Utilisation : Code d'état final pour les opérations après traitement.

Valeurs Possibles : Identifiants numériques définis dans le système.

`CETAOPEALLICOM1` :

Utilisation : Code d'état initial pour les opérations avant traitement.

Valeurs Possibles : Identifiants numériques définis dans le système.
