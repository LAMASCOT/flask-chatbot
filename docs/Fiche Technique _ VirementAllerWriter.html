Introduction

Description : La tache VirementAllerWriter est une extension de FlatFileWriter utilisée pour générer et envoyer des fichiers de virements vers un système externe. Elle traite les virements, met à jour leur état, et génère un fichier plat contenant les détails des virements.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Générer et envoyer des fichiers de virements, en mettant à jour les états des virements et en générant un fichier plat avec les détails des virements.

Traitement :

Met à jour l'état des virements dans la base de données.

Récupère les virements à traiter et calcule le montant total.

Génère un fichier plat contenant les détails des virements.

Met à jour l'état des virements après traitement.

Méthodes

execute() : Exécute le processus de génération et d'envoi des fichiers de virements, en mettant à jour les états des virements, en générant le fichier plat, et en enregistrant les événements dans les logs.

Paramètres Utilisés

CETAOPEVALDELTA : Paramètre utilisé pour identifier l'état des virements à traiter.

CETAOPEALLICOM1 : Paramètre utilisé pour mettre à jour l'état des virements après traitement.

CODE_BANQUE : Code de la banque utilisé pour identifier l'établissement.

CODE_BANQUE_SICA3 : Code de la banque utilisé pour identifier l'établissement dans le fichier.

SIB_IN_FOLDER : Répertoire de destination pour le fichier de virements généré.

Remarques

Gestion des Erreurs : Les erreurs lors du traitement des virements et de la génération des fichiers sont enregistrées dans les logs.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de vérifications ou de formats de fichiers si nécessaire.

Configuration : La classe utilise des composants configurables pour le traitement des virements et la génération des fichiers, permettant une adaptation facile aux besoins spécifiques du système.

Fichier Plat : Le fichier plat généré contient les détails des virements, y compris le nombre total de virements et le montant total, et est enregistré dans le répertoire spécifié par SIB_IN_FOLDER.
