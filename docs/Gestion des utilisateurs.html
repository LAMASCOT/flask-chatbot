Pour respecter la politique de sécurité de la banque, vous optez pour la connexion à votre annuaire Active Directory. L’interfaçage avec celui-ci est réalisé via les éléments suivants :

  • Domaine racine de l’annuaire (exemple dc =smartaps, dc =tech)
    Port spécifique (389 par défaut)

  • Adresse ip ou nom d’hôte de l’AD

  • Utilisateur ayant droit en lecture seule sur l’annuaire
