Introduction

Description : La tache Cro135T24OBAMoveForUpload est une extension de FlatFileWriter utilisée pour déplacer des fichiers CRO 135 vers un répertoire de destination spécifique pour le système T24. Elle assure la gestion des fichiers en les déplaçant et en les sauvegardant si nécessaire.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Déplacer les fichiers CRO 135 vers un répertoire de destination pour le système T24.

Traitement :

Récupère les fichiers CRO 135 à partir d'un répertoire source.

Déplace les fichiers vers un répertoire de destination.

Sauvegarde les fichiers existants avant de les écraser.

Supprime les fichiers sources après le déplacement.

Méthodes

Cro135T24OBAMoveForUpload() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de déplacement des fichiers CRO 135.

moveAndOverwrite(File source, File dest) : Déplace un fichier source vers une destination, en sauvegardant le fichier de destination s'il existe déjà.

getNonExistingTempFile(File inputFile) : Génère un nom de fichier temporaire qui n'existe pas dans le même répertoire que le fichier d'entrée.

end() : Termine le processus en enregistrant un événement de log.

Paramètres Utilisés

SIB_IN_FOLDER : Paramètre indiquant le répertoire source des fichiers CRA.

DEST_CRA_FOLDER : Paramètre indiquant le répertoire de destination pour les fichiers CRO.

Remarques

Gestion des Erreurs : Les erreurs lors du déplacement ou de la suppression des fichiers sont gérées par des logs.

Extensibilité : La classe peut être étendue pour inclure d'autres types de fichiers ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons IO : La classe utilise FileUtils de la bibliothèque Apache Commons IO pour la manipulation des fichiers.
