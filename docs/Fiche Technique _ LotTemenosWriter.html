Introduction

Description : La tache LotTemenosWriter est une extension de FlatFileWriter utilisée pour gérer l'envoi des lots de chèques et d'effets vers le système Temenos. Elle traite les lots de chèques et d'effets, en les préparant pour l'envoi sous forme de fichiers plats.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Préparer et envoyer des lots de chèques et d'effets vers le système Temenos.

Traitement :

Récupère les chèques et les effets depuis la base de données.

Génère des fichiers plats contenant les informations des chèques et des effets.

Met à jour l'état des chèques, des effets et des remises après traitement.

Méthodes

LotTemenosWriter() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de préparation et d'envoi des chèques et des effets.

prepareCheques(List<Cheques> cheques) : Prépare les chèques pour l'envoi.

prepareEffets(List<Effets> effets, String source) : Prépare les effets pour l'envoi.

printLotCheque(List<Cheques> cheques, EnteteLot enteteLot) : Génère le fichier plat pour un lot de chèques.

printLotEffet(List<Effets> effets, EnteteLot enteteLot, String source) : Génère le fichier plat pour un lot d'effets.

updateEffets(List<Effets> allEffets) : Met à jour l'état des effets après traitement.

addErrorMessage(String message, String fromEcran) : Ajoute un message d'erreur.

addInfoMessage(String message, String fromEcran) : Ajoute un message d'information.

addWarningMessage(String message, String fromEcran) : Ajoute un message d'avertissement.

printEnteteLot(EnteteLot enteteLot, String source) : Génère l'en-tête du fichier pour un lot.

Paramètres Utilisés

CETAOPEVAL : Paramètre indiquant l'état des opérations validées.

CETAOPEVAL2 : Paramètre indiquant un autre état des opérations validées.

CETAOPESAI : Paramètre indiquant l'état des opérations saisies.

CETAOPEALLICOM1 : Paramètre indiquant l'état des opérations prêtes pour l'envoi.

CETAOPEVALSURCAIENVSIB : Paramètre indiquant l'état des opérations envoyées au SIB.

Remarques

Gestion des Erreurs : Les erreurs lors de la préparation des chèques et des effets sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe peut être étendue pour inclure d'autres types de chèques ou d'effets, ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
