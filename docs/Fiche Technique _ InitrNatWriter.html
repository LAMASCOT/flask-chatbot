Introduction

Description : La tache InitrNatWriter est une extension de FlatFileWriter utilisée pour générer et envoyer des fichiers d'initialisation nationale (INITR) vers la BCEAO. Elle initialise la journée de compensation en créant un fichier spécifique avec les informations nécessaires.

Package

Emplacement : sas.action.writers.sica

Description Fonctionnelle

Objectif : Générer et envoyer un fichier d'initialisation nationale vers la BCEAO pour démarrer la journée de compensation.

Traitement :

Récupère les paramètres nécessaires pour l'initialisation.

Crée un en-tête de remise avec les informations de l'émetteur, du récepteur, et d'autres détails.

Génère un fichier avec les informations d'initialisation et l'envoie à la BCEAO.

Méthodes

InitrNatWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus de génération et d'envoi du fichier d'initialisation nationale.

Paramètres Utilisés

VERSION_SICA : Version du système SICA utilisé.

INITR_NAT : Compteur pour les fichiers d'initialisation nationale.

MONITOR_LINES : Nombre de lignes à conserver dans le journal des événements.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération ou de l'envoi du fichier sont gérées par des logs et des messages d'erreur.

Extensibilité : La tache peut être étendue pour inclure d'autres types d'initialisations ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et gérer les paramètres de l'application.
