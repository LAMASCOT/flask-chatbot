Introduction

Description : La tache MTSFDWriter est une extension de FlatFileWriter utilisée pour générer des fichiers de messages SWIFT MT103 et MT202 pour les Systèmes Financiers Décentralisés (SFD). Elle traite les virements RTGS (Real-Time Gross Settlement) et génère les fichiers nécessaires pour les transactions interbancaires.

Package

Emplacement : sas.action.writers.corporates.sfd.rtgs

Description Fonctionnelle

Objectif : Générer des fichiers de messages SWIFT MT103 et MT202 pour les transactions RTGS.

Traitement :

Récupère les virements RTGS de la base de données.

Prépare et génère les fichiers MT103 et MT202 en fonction des types de virements.

Écrit les fichiers générés dans le répertoire spécifié.

Attributs

db : DataBaseJPA - Instance pour l'accès à la base de données.

parameters : ConcurrentHashMap<String, Object> - Paramètres pour les requêtes SQL.

mt103Virs : List<VirementsRTGS> - Liste des virements de type MT103.

mt202SfdToBankVirs : List<VirementsRTGS> - Liste des virements de type MT202 de SFD vers Banque.

mt202SfdToSfdVirs : List<VirementsRTGS> - Liste des virements de type MT202 de SFD vers SFD.

Méthodes

MTSFDWriter() : Constructeur par défaut qui initialise les attributs nécessaires.

execute() : Exécute le processus de génération des fichiers de messages SWIFT.

prepareVirMT103(List<VirementsRTGS> vir) : Prépare et génère les fichiers pour les virements de type MT103.

prepareVirMT202ToBank(List<VirementsRTGS> vir) : Prépare les virements de type MT202 de SFD vers Banque.

prepareVirMT202ToSfd(List<VirementsRTGS> vir) : Prépare les virements de type MT202 de SFD vers SFD.

writeMT103File(VirementsRTGS virMT103) : Génère et écrit un fichier MT103 pour un virement donné.

serializeToMTFin(String fileName, String document) : Sérialise le message SWIFT dans un fichier.

Paramètres Utilisés

CETAOPEVALDELTA : État des opérations à évaluer, utilisé pour filtrer les virements.

SIB_IN_FOLDER : Répertoire de destination pour les fichiers générés.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs.

Extensibilité : La classe est conçue pour être extensible, permettant l'ajout de nouveaux types de messages ou de fonctionnalités.

Utilisation des Bibliothèques SWIFT : Utilise les bibliothèques Prowide pour manipuler les messages SWIFT.
