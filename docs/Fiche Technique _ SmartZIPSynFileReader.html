Introduction

Description : La tache SmartZIPSynFileReader est une extension de FlatFileReader utilisée pour traiter les fichiers ZIP synchronisés. Elle gère l'extraction des fichiers contenus dans les archives ZIP et leur traitement ultérieur.

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers ZIP dans le répertoire configuré.

Traitement : Extrait les fichiers contenus dans l'archive ZIP, puis traite chaque fichier extrait en utilisant un lecteur de fichiers spécifique.

Résultat : Les fichiers extraits sont traités et les archives ZIP peuvent être sauvegardées ou supprimées selon la configuration.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers ZIP : Assure l'extraction des fichiers contenus dans les archives ZIP.

Interaction avec la Base de Données : Met à jour les informations pertinentes après le traitement des fichiers extraits.

Traitement des Fichiers Extraits : Utilise SmartImageChequeSynReader pour traiter les fichiers extraits.

Vérification de Conformité : Vérifie si les fichiers ZIP ont déjà été traités pour éviter les doublons.

Paramètres du Composant

SAVEZIPFILE : Indicateur pour sauvegarder ou supprimer les fichiers ZIP après traitement.

workFolder : Répertoire de travail temporaire pour l'extraction des fichiers.

zipFolder : Répertoire de sauvegarde pour les fichiers ZIP traités.
