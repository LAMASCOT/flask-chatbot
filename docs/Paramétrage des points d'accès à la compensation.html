PARAMETRAGE DES POINTS D’ACCES A LA COMPENSATION

Introduction

Ce paramétrage permet de configurer les points d’accès utilisés par l’application pour transmettre les opérations de compensation bancaire, que ce soit au niveau sous-régional ou national. Ces paramètres sont essentiels pour diriger correctement les flux de compensation vers les systèmes correspondants (comme SICA-UEMOA pour la sous-région). Ils assurent la conformité des opérations de règlement avec les standards des autorités monétaires et facilitent une meilleure gestion des circuits interbancaires.

Les points d’accès sont définis en fonction de la localisation géographique et du type de compensation, afin de garantir un traitement efficace, structuré et automatisé des opérations financières.

Point d’accès à la compensation sous-régionale

  • Nom du paramètre : PAC_SCSR_SICA3

  • Valeur par défaut : CI0

  • Libellé : POINT D’ACCÈS À LA COMPENSATION SOUS-RÉGIONALE

  • Type : CODE_PAC

Description :
Ce paramètre indique le point d’accès technique utilisé pour acheminer les opérations vers la plateforme de compensation sous-régionale, généralement SICA-UEMOA. La valeur CI0 correspond au code du point d’accès configuré pour la Côte d’Ivoire dans le cadre de la compensation sous-régionale.
Il permet de diriger les flux interbancaires qui concernent plusieurs pays membres de l’Union vers l’infrastructure régionale appropriée. Ce point d’accès est essentiel pour toutes les transactions transfrontalières dans la zone UEMOA.

Point d’accès à la compensation nationale

  • Nom du paramètre : PAC_SCMP_SICA3

  • Valeur par défaut : CI1

  • Libellé : Point d’accès à la compensation nationale

  • Type : CODE_PAC

Description :
Ce paramètre définit le point d’accès réservé aux opérations de compensation traitées uniquement au niveau national. La valeur CI1 est associée à la Côte d’Ivoire pour la compensation locale.
Il permet de séparer les flux domestiques des flux sous-régionaux, en dirigeant les opérations vers l’infrastructure dédiée à la gestion des échanges interbancaires internes. Ce point d’accès est particulièrement utile pour optimiser le traitement des chèques, virements ou autres moyens de paiement destinés aux établissements locaux.
