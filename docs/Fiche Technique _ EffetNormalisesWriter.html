Introduction

Description : La tache EffetNormalisesWriter est une extension de FlatFileWriter utilisée pour gérer l'écriture des effets normalisés. Elle interagit avec la base de données pour récupérer et traiter les données nécessaires à l'écriture des fichiers plats.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Gérer l'écriture des effets normalisés en interagissant avec la base de données et en générant des fichiers plats.

Traitement :

Utilise DataBaseJPA pour interagir avec la base de données.

Utilise ConcurrentHashMap pour gérer les paramètres nécessaires aux opérations de base de données.

Attributs

dataBaseJPA : Instance de DataBaseJPA utilisée pour les opérations de base de données.

parameters : ConcurrentHashMap pour stocker les paramètres utilisés dans les requêtes de base de données.

Remarques

Extensibilité : La tache est conçue pour être étendue avec des méthodes spécifiques pour traiter les effets normalisés.

Intégration : Cette tache est intégrée dans le package sas.action.writers.orion et utilise les utilitaires de patware pour la gestion des fichiers et des bases de données.
