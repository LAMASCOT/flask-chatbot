Introduction

Description : La tache LotaSFDWriter est une extension de FlatFileWriter utilisée pour générer des fichiers LOTA destinés à être envoyés à un partenaire. Elle est responsable de la création et de l'écriture des fichiers LOTA basés sur les informations fournies.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Générer et envoyer des fichiers LOTA vers un partenaire.

Traitement :

Génère le nom du fichier LOTA en fonction des paramètres fournis.

Écrit les informations de lot dans le fichier LOTA.

Ferme le fichier après l'écriture.

Attributs

nomFichierLota : String - Nom du fichier LOTA généré.

codeReponse : String - Code de réponse pour le fichier LOTA.

banqueRemettante : String - Code de la banque remettante.

numero : String - Numéro associé au fichier LOTA.

nbOperations : String - Nombre d'opérations incluses dans le fichier LOTA.

montanTotal : String - Montant total des opérations dans le fichier LOTA.

typeSytemeCompensation : String - Type de système de compensation utilisé.

idSFD : String - Identifiant du SFD (Système Financier Décentralisé).

Méthodes

LotaSFDWriter() : Constructeur par défaut qui initialise la description du processus.

LotaSFDWriter(String name) : Constructeur qui initialise le nom du fichier LOTA en fonction du nom fourni.

LotaSFDWriter(String codeReponse, String banqueRemettante, String numero, String nbOperations, String montanTotal, String typeSytemeCompensation, String idSFD) : Constructeur qui initialise les attributs du fichier LOTA.

getNomFichierLota() : Retourne le nom du fichier LOTA.

setNomFichierLota(String nomFichierLota) : Définit le nom du fichier LOTA.

execute() : Exécute le processus de génération du fichier LOTA.

getCodeReponse() : Retourne le code de réponse.

setCodeReponse(String codeReponse) : Définit le code de réponse.

getBanqueRemettante() : Retourne le code de la banque remettante.

setBanqueRemettante(String banqueRemettante) : Définit le code de la banque remettante.

getNumero() : Retourne le numéro associé au fichier LOTA.

setNumero(String numero) : Définit le numéro associé au fichier LOTA.

getNbOperations() : Retourne le nombre d'opérations.

setNbOperations(String nbOperations) : Définit le nombre d'opérations.

getMontanTotal() : Retourne le montant total des opérations.

setMontanTotal(String montanTotal) : Définit le montant total des opérations.

getTypeSytemeCompensation() : Retourne le type de système de compensation.

setTypeSytemeCompensation(String typeSytemeCompensation) : Définit le type de système de compensation.

getIdSFD() : Retourne l'identifiant du SFD.

setIdSFD(String idSFD) : Définit l'identifiant du SFD.

Paramètres Utilisés

SIB_IN_FOLDER_SFD : Répertoire de destination pour les fichiers LOTA générés.

DEVISE : Devise utilisée pour les opérations.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de fichiers ou formats.

Coordination : La classe LotaSFDWriter coordonne la génération des fichiers LOTA, assurant une gestion efficace des transferts pour les partenaires.
