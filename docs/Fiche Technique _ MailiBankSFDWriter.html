Introduction

Description : La tache MailiBankSFDWriter est une extension de FlatFileWriter utilisée pour orchestrer la génération des fichiers MAILI vers la banque. Elle utilise deux autres classes, MailiNatBankSFDWriter et MailiSrgBankSFDWriter, pour gérer respectivement les fichiers nationaux et sous-régionaux.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Orchestrer la génération des fichiers MAILI pour les envois vers la banque, en combinant les fichiers nationaux et sous-régionaux.

Traitement :

Initialise les instances de MailiNatBankSFDWriter et MailiSrgBankSFDWriter.

Exécute la génération des fichiers pour les deux types de transactions (nationales et sous-régionales).

Met à jour la description du processus pour inclure les résultats des deux opérations.

Attributs

mailiNatBankSFDWriter : MailiNatBankSFDWriter - Instance utilisée pour gérer la génération des fichiers nationaux.

mailiSrgBankSFDWriter : MailiSrgBankSFDWriter - Instance utilisée pour gérer la génération des fichiers sous-régionaux.

Méthodes

MailiBankSFDWriter() : Constructeur par défaut qui initialise les descriptions et les instances des classes de gestion des fichiers.

execute() : Exécute le processus de génération des fichiers MAILI, en appelant les méthodes d'exécution des instances MailiNatBankSFDWriter et MailiSrgBankSFDWriter, et en mettant à jour la description du processus.

Paramètres Utilisés

Aucun paramètre spécifique mentionné dans le code fourni : La classe utilise des méthodes pour passer des paramètres via setParametersMap() aux instances de MailiNatBankSFDWriter et MailiSrgBankSFDWriter.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de transactions ou formats de fichiers.

Coordination : La classe MailiBankSFDWriter coordonne la génération des fichiers MAILI pour les transactions nationales et sous-régionales, assurant une gestion efficace des transferts pour les banques.
