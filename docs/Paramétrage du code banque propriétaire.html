PARAMETRAGE DU CODE BANQUE PROPRIETAIRE

Introduction

Ce paramétrage permet de spécifier le code d’identification de la banque dans les systèmes de compensation et dans les échanges interbancaires. Il est essentiel pour assurer une identification unique et correcte de la banque lors des traitements automatisés, que ce soit pour les opérations nationales ou sous-régionales.
Les systèmes comme SICA (Système Interbancaire de Compensation Automatisé) exigent que chaque établissement participant dispose d’un code unique et conforme aux normes définies par les autorités monétaires.

Code Banque dans le système SICA3

  • Nom du paramètre : CODE_BANQUE_SICA3

  • Valeur par défaut : SN140

  • Libellé : Code Banque Propriétaire

  • Type : CODE_BANQUE

Description :
Ce paramètre représente le code unique attribué à la banque dans le système de compensation sous-régionale SICA3. La valeur SN140 identifie l’établissement bancaire au sein de cette infrastructure régionale.
Ce code est utilisé dans tous les échanges et règlements interbancaires transitant par la plateforme SICA, assurant ainsi une traçabilité et une bonne reconnaissance de la banque par les autres institutions financières.

Code Banque (général)

  • Nom du paramètre : CODE_BANQUE

  • Valeur par défaut : SN140

  • Libellé : Code Banque Propriétaire

  • Type : CODE_BANQUE

Description :
Ce paramètre désigne le code d’identification général de la banque dans les différents systèmes de traitement bancaire internes et externes. Il est utilisé à travers l’ensemble des modules de l’application pour représenter l’entité bancaire propriétaire de la plateforme.
Le code SN140 est utilisé comme identifiant principal de la banque dans les opérations internes, les échanges de fichiers, les rapports, et les interfaces avec d’autres systèmes tiers.
