Introduction

Description : La tache PrelevementAllerWriter est une extension de FlatFileWriter utilisée pour générer des fichiers de prélèvements aller. Elle traite les prélèvements valides, génère un fichier plat contenant les détails des prélèvements, et enregistre ces fichiers dans un répertoire spécifié.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Générer des fichiers de prélèvements aller, en traitant les prélèvements valides et en générant un fichier plat avec les détails des prélèvements.

Traitement :

Met à jour l'état des prélèvements valides dans la base de données.

Récupère les prélèvements à traiter.

Calcule le montant total des prélèvements.

Génère un fichier plat contenant les détails des prélèvements.

Met à jour l'état des prélèvements après traitement.

Enregistre les fichiers générés dans le répertoire spécifié par SIB_IN_FOLDER.

Méthodes

execute() : Exécute le processus de génération des fichiers de prélèvements aller, en mettant à jour l'état des prélèvements, en récupérant les prélèvements à traiter, en générant les fichiers plats, et en enregistrant les événements dans les logs.

Paramètres Utilisés

CETAOPEVALDELTA : Paramètre utilisé pour identifier l'état des prélèvements valides.

CETAOPEALLICOM1 : Paramètre utilisé pour identifier l'état des prélèvements après traitement.

SIB_IN_FOLDER : Répertoire de destination pour les fichiers de prélèvements générés.

CODE_BANQUE_SICA3 : Code de la banque utilisé dans le fichier plat.

CODE_BANQUE : Code de la banque utilisé dans le nom du fichier.

Remarques

Gestion des Erreurs : Les erreurs lors du traitement des prélèvements et de la génération des fichiers sont enregistrées dans les logs.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de vérifications ou de formats de fichiers si nécessaire.

Configuration : La classe utilise des composants configurables pour le traitement des prélèvements et la génération des fichiers, permettant une adaptation facile aux besoins spécifiques du système.

Fichier Plat : Le fichier plat généré contient les détails des prélèvements, y compris le montant total, le nombre de prélèvements, et d'autres informations pertinentes, et est enregistré dans le répertoire spécifié par SIB_IN_FOLDER.
