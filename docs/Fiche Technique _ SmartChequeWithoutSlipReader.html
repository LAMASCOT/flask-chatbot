SmartChequeWithoutSlipReader

Introduction

Description : La tache SmartChequeWithoutSlipReader est conçue pour lire, traiter et sauvegarder les fichiers de chèques scannés sans bordereau. Elle intègre les informations dans le système et applique des vérifications de conformité pour assurer l'intégrité des données.

Description Fonctionnelle

Déclenchement : La classe est activée lors de la détection de nouveaux fichiers de chèques scannés dans le répertoire configuré.

Traitement : Elle vérifie les données des fichiers scannés, traite les images des chèques, applique des tatouages numériques, et met à jour les informations dans la base de données.

Résultat : Les statuts des chèques et des séquences sont mis à jour dans la base de données, et les images des chèques sont déplacées vers le répertoire approprié.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers : La classe assure la lecture, la validation, et le déplacement des fichiers de chèques scannés.

Interaction avec la Base de Données : Elle met à jour les statuts des chèques et des séquences.

Traitement d'Images : Elle redimensionne, tatoue et déplace les images des chèques.

Vérification des Codes SICA : La classe intègre la conversion des codes numériques en codes alphanumériques pour les banques.

Paramètres du Composant

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées.

CODE_SICA2 et CODE_SICA3 : Paramètres pour la conversion des codes numériques en codes alphanumériques.

CETAOPESTO et CETAIMASTO : Statuts utilisés pour les opérations et les images des chèques.

Exemples d'Utilisation

La classe est utilisée pour valider, traiter et sauvegarder les chèques scannés sans bordereau dans un environnement bancaire.

Intégration avec d'autres Systèmes

Elle est intégrée avec le système de gestion des chèques, les services de tatouage numérique, et les bases de données pour la gestion des séquences et des chèques.
