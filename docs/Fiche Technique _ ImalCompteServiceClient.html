Introduction

Description : La tache ImalCompteServiceClient est un client de service REST utilisé pour interagir avec le système IMAL. Elle permet de récupérer les informations de compte via des appels API RESTful.

Package

Emplacement : sas.action.writers.forimal

Description Fonctionnelle

Objectif : Fournir une interface pour interagir avec le service REST d'IMAL afin de récupérer les informations de compte.

Traitement :

Configure un client REST pour se connecter au service IMAL.

Envoie des requêtes pour récupérer les informations de compte.

Gère les erreurs potentielles lors des appels API.

Méthodes

ImalCompteServiceClient() : Constructeur qui initialise le client REST et configure les paramètres de connexion.

returnCompte(CompteRequest request) : Envoie une requête pour récupérer les informations de compte et retourne la réponse sous forme d'un objet ComptesResponse.

returnMapper() : Crée et configure un ObjectMapper pour la sérialisation et la désérialisation JSON.

close() : Ferme le client REST pour libérer les ressources.

Attributs

LOG : Logger pour enregistrer les événements et les erreurs.

fetchCompte : Cible Web pour les requêtes de récupération de compte.

client : Client REST configuré pour les appels API.

BASE_URI : URI de base pour le service REST d'IMAL, récupéré via Utility.getParam("IMAL_REST_URI").

ACCOUNT_REL_URI : URI relatif pour accéder aux informations de compte.

dataBaseJPA : Instance de DataBaseJPA pour interagir avec la base de données.

Paramètres Utilisés

IMAL_REST_URI : URI de base pour le service REST d'IMAL, utilisé pour configurer le client REST.

Remarques

Gestion des Erreurs : Les erreurs lors des appels API sont capturées et traitées, avec des codes d'erreur spécifiques définis dans GenericError.

Extensibilité : La tache peut être étendue pour inclure d'autres types de requêtes ou de services REST.

Intégration : Conçue pour être utilisée dans des contextes où les informations de compte doivent être récupérées depuis le système IMAL via des appels API RESTful.
