Introduction

Description : La tache GenOrdersBDABankForSFDWriter est une extension de FlatFileWriter utilisée pour générer des fichiers d'ordres pour les Systèmes Financiers Décentralisés (SFD) à destination des banques. Elle utilise l'écrivain de fichiers LotBDASFDWriter pour produire les fichiers nécessaires.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Générer des fichiers d'ordres pour les SFD vers les banques en utilisant LotBDASFDWriter.

Traitement :

Initialise et exécute l'écrivain de fichiers LotBDASFDWriter.

Met à jour la description du processus pour refléter l'activité de génération des fichiers.

Attributs

lotSFDBankWriter : LotBDASFDWriter - Instance de l'écrivain de fichiers pour les lots de SFD.

Méthodes

GenOrdersBDABankForSFDWriter() : Constructeur par défaut qui initialise l'écrivain de fichiers et la description du processus.

execute() : Exécute le processus de génération des fichiers d'ordres en utilisant LotBDASFDWriter.

Paramètres Utilisés

ParametersMap : Utilisé pour passer des paramètres de configuration à l'écrivain de fichiers LotBDASFDWriter.

Remarques

Coordination : La classe GenOrdersBDABankForSFDWriter agit comme un coordinateur, orchestrant l'exécution de l'écrivain de fichiers LotBDASFDWriter pour assurer la génération complète des fichiers d'ordres pour les SFD.

Extensibilité : La classe est conçue pour être facilement extensible, permettant l'ajout de nouveaux écrivains de fichiers ou de nouvelles fonctionnalités si nécessaire.

Gestion des Erreurs : Les erreurs lors de l'exécution de l'écrivain de fichiers sont gérées par LotBDASFDWriter, et la description du processus est mise à jour en conséquence.
