Introduction

Description : La tache ChequeAllerBNIWriter est une extension de FlatFileWriter utilisée pour envoyer des chèques vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques valides vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Récupère les chèques valides à envoyer.

Génère un fichier plat avec les informations des chèques.

Met à jour l'état des chèques après envoi.

Méthodes

ChequeAllerBNIWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des chèques vers le SIB.

prepareCheques(List<Cheques> cheques) : Prépare les chèques pour l'envoi en générant les fichiers plats.

printLotCheque(List<Cheques> cheques, EnteteLot enteteLot) : Imprime les informations des chèques dans le fichier plat.

updateEtatOperations() : Met à jour l'état des chèques après envoi.

Paramètres Utilisés

CETAOPEVAL2 : État des chèques valides pour envoi.

CETAOPEALLICOM1 : État des chèques après envoi.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et effectuer les mises à jour nécessaires.
