Introduction

Description : La tache ChequeAllerCIEWriter est une extension de FlatFileWriter utilisée pour envoyer des chèques "Aller" vers le Système d'Information Bancaire (SIB). Elle récupère les chèques à partir de la base de données, les traite, et les insère dans une table de compensation.

Package

Emplacement : sas.action.writers.cie

Description Fonctionnelle

Objectif : Envoyer les chèques "Aller" vers le SIB pour traitement.

Traitement :

Récupère les chèques avec un état spécifique depuis la base de données.

Insère les chèques dans la table IBUS_CHQ_COMP pour la compensation.

Met à jour l'état des chèques après insertion réussie.

Attributs

db : DataBase - Instance pour l'accès à la base de données.

Méthodes

ChequeAllerCIEWriter() : Constructeur par défaut qui initialise la description du processus.

execute() : Exécute le processus d'envoi des chèques "Aller" vers le SIB.

Paramètres Utilisés

CETAOPEALLICOM1ACC : État des chèques à récupérer pour l'envoi.

CETAOPEALLICOM1ACCENVSIB : Nouvel état à attribuer aux chèques après envoi réussi.

Remarques

Gestion des Erreurs : Les erreurs lors de l'insertion des chèques sont gérées par des logs, et un message d'avertissement est enregistré en cas d'échec.

Extensibilité : La classe est conçue pour être extensible, permettant l'ajout de nouvelles fonctionnalités ou l'intégration avec d'autres systèmes de compensation.

Utilisation de la Base de Données : Utilise DataBase et JDBCXmlReader pour gérer les connexions et les opérations sur la base de données.
