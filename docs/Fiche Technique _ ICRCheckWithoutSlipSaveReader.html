ICRCheckWithoutSlipSaveReader

Introduction

Description : La tache ICRCheckWithoutSlipSaveReader est conçue pour lire, traiter et sauvegarder les fichiers de chèques scannés sans bordereau, en intégrant les informations dans le système et en appliquant des vérifications de conformité.

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de chèques scannés dans le répertoire configuré.

Traitement : Vérifie les données des fichiers scannés, traite les images des chèques, applique des tatouages numériques, et met à jour les informations dans la base de données.

Résultat : Mise à jour des statuts des chèques et des séquences dans la base de données, avec les images des chèques déplacées vers le répertoire approprié.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers : Assure la lecture, la validation, et le déplacement des fichiers de chèques scannés.

Interaction avec la Base de Données : Met à jour les statuts des chèques et des séquences.

Traitement d'Images : Redimensionne, tatoue et déplace les images des chèques.

Vérification ICR : Intègre les services ICR pour la reconnaissance des caractères et la validation des informations de chèque.

Paramètres du Composant

LINE_SCORE_PARAM : Paramètre de score de ligne utilisé pour la validation.

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées.

CODE_SICA2 et CODE_SICA3 : Paramètres pour la conversion des codes numériques en codes alphanumériques.

ICR_ACTIVATED : Indique si le service ICR est activé.

ICR_VERSION : Version du service ICR utilisé.

Exemples d'Utilisation

Utilisée pour valider, traiter et sauvegarder les chèques scannés sans bordereau dans un environnement bancaire.

Intégration avec d'autres Systèmes

Intégrée avec le système de gestion des chèques, le service ICR pour la reconnaissance des caractères, et les services de tatouage numérique.
