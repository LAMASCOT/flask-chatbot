Introduction

Description : La tache SmartSFTPUploader est conçue pour gérer le transfert de fichiers via SFTP. Elle assure la connexion sécurisée à un serveur distant et le transfert de fichiers tout en gérant les erreurs et les tentatives de reconnexion.

Description Fonctionnelle

Déclenchement : La tache est activée pour transférer des fichiers vers un serveur SFTP spécifié.

Traitement : Elle établit une connexion SFTP, transfère le fichier, et gère les erreurs potentielles avec des tentatives de reconnexion.

Résultat : Les fichiers sont transférés avec succès vers le serveur SFTP, et une copie est archivée localement.

En cas d’erreur : Les erreurs sont consignées dans les journaux, et des tentatives de reconnexion sont effectuées si configurées.

Opérations Clés

Gestion des Connexions SFTP : Établit et gère les connexions SFTP, y compris l'authentification par mot de passe ou clé privée.

Transfert de Fichiers : Transfère les fichiers vers le répertoire distant spécifié sur le serveur SFTP.

Archivage Local : Copie les fichiers transférés dans un répertoire d'archives local pour la traçabilité.

Gestion des Erreurs et Reconnexions : Gère les erreurs de connexion et effectue des tentatives de reconnexion si nécessaire.

Paramètres du Composant

SFTP_TIME_ALIVE : Intervalle de temps pour garder la connexion SFTP active, avec une valeur par défaut de 300000 ms.

DEBUG_PASS : Indicateur pour activer le débogage des mots de passe, avec une valeur par défaut false.

CRYPT_TYPE : Type de cryptage utilisé pour les mots de passe, avec une valeur par défaut AES-256.

PREFERRED_AUTH_SFTP : Méthode d'authentification préférée pour SFTP, avec une valeur par défaut password.

SFTP_RETRY : Indicateur pour activer les tentatives de reconnexion, avec une valeur par défaut true.

SFTP_OLD_ALGOS : Indicateur pour utiliser d'anciens algorithmes de chiffrement, avec une valeur par défaut false.

SFTP_CONN_VIA_KEY : Indicateur pour l'utilisation de l'authentification par clé privée, avec une valeur par défaut false.

Exemples d'Utilisation

Utilisée pour transférer des fichiers de manière sécurisée vers un serveur SFTP dans un environnement bancaire ou d'entreprise.

Intégration avec d'autres Systèmes

Intégrée avec les systèmes de gestion de fichiers et de sécurité pour assurer un transfert sécurisé et fiable des fichiers.
