Introduction

Description : La tache MailiSFDWriter est une extension de FlatFileWriter utilisée pour orchestrer la génération des fichiers MAILI vers la banque. Elle utilise deux autres classes, MailiNatSFDWriter et MailiSrgSFDWriter, pour gérer respectivement les fichiers nationaux et sous-régionaux.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Orchestrer la génération des fichiers MAILI pour les envois vers la banque, en combinant les fichiers nationaux et sous-régionaux.

Traitement :

Initialise les instances de MailiNatSFDWriter et MailiSrgSFDWriter.

Exécute la génération des fichiers pour les deux types de transactions (nationales et sous-régionales).

Met à jour la description du processus pour inclure les résultats des deux opérations.

Attributs

mailiNatSFDWriter : MailiNatSFDWriter - Instance utilisée pour gérer la génération des fichiers nationaux.

mailiSrgSFDWriter : MailiSrgSFDWriter - Instance utilisée pour gérer la génération des fichiers sous-régionaux.

Méthodes

MailiSFDWriter() : Constructeur par défaut qui initialise les instances de MailiNatSFDWriter et MailiSrgSFDWriter et définit la description du processus.

execute() : Exécute le processus de génération des fichiers MAILI, en appelant les méthodes execute() des instances mailiNatSFDWriter et mailiSrgSFDWriter, et met à jour la description du processus avec les résultats.

Composants Internes

MailiNatSFDWriter, MailiSrgSFDWriter : Classes utilisées pour gérer respectivement les fichiers nationaux et sous-régionaux.

Paramètres Utilisés

Parameters Map : Utilisé pour passer les paramètres de configuration aux instances de MailiNatSFDWriter et MailiSrgSFDWriter.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par les classes MailiNatSFDWriter et MailiSrgSFDWriter.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de transactions ou formats de fichiers.

Configuration : La classe utilise une map de paramètres pour configurer les instances de MailiNatSFDWriter et MailiSrgSFDWriter, permettant une adaptation facile aux besoins spécifiques du système.

Coordination : La classe MailiSFDWriter coordonne la génération des fichiers MAILI pour les transactions nationales et sous-régionales, assurant une gestion efficace des transferts vers la banque.
