Introduction

Description : La tache EffetsAllerTemenosRemisesWriter est une extension de FlatFileWriter utilisée pour gérer l'envoi des effets vers le système Temenos. Elle traite les remises d'effets, en les préparant pour l'envoi sous forme de fichiers plats.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Préparer et envoyer des remises d'effets vers le système Temenos.

Traitement :

Récupère les effets associés à une remise depuis la base de données.

Génère un fichier plat contenant les informations des effets.

Met à jour l'état des effets après traitement.

Méthodes

EffetsAllerTemenosRemisesWriter() : Constructeur par défaut qui initialise la description de l'opération.

EffetsAllerTemenosRemisesWriter(Remises remise) : Constructeur qui initialise la remise à traiter.

execute() : Exécute le processus de préparation et d'envoi des effets.

getRemise() : Retourne la remise en cours de traitement.

setRemise(Remises remise) : Définit la remise à traiter.

Paramètres Utilisés

CETAOPEERR : Paramètre indiquant l'état des opérations en erreur.

CETAOPEVAL : Paramètre indiquant l'état des opérations validées.

CETAREMENVSIB : Paramètre indiquant l'état des remises envoyées au SIB.

Remarques

Gestion des Erreurs : Les erreurs lors de la préparation des effets sont gérées par des logs.

Extensibilité : La classe peut être étendue pour inclure d'autres types de remises ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
