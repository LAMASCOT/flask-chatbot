Introduction

Description : La tache ChequeAllerSitWriter est une extension de FlatFileWriter utilisée pour envoyer des chèques vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques valides vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Vérifie si le verrouillage pour l'envoi des chèques est activé.

Récupère les chèques valides à envoyer.

Génère un fichier plat avec les informations des chèques.

Met à jour l'état des chèques et des remises après envoi.

Méthodes

ChequeAllerSitWriter() : Constructeur par défaut qui initialise la description de l'envoi et les états de début et de fin des chèques.

execute() : Exécute le processus d'envoi des chèques vers le SIB.

Paramètres Utilisés

LOT_CHEQUE_ETAT_DEBUT : État initial des chèques pour envoi.

LOT_CHEQUE_ETAT_FIN : État final des chèques après envoi.

CETAOPEVAL : État des chèques valides pour envoi.

CETAOPEALLICOM1 : État des chèques après envoi.

CETAREMENVSIB : État des remises après envoi.

LOCK_CHK_ALLER : Paramètre de verrouillage pour éviter les envois simultanés.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et effectuer les mises à jour nécessaires.
