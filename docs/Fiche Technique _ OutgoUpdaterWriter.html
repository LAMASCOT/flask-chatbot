Introduction

Description : La tache OutgoUpdaterWriter est une extension de FlatFileWriter utilisée pour mettre à jour l'état des chèques dans la base de données. Elle modifie l'état des chèques en fonction de critères spécifiques et enregistre les motifs de rejet.

Package

Emplacement : sas.action.writers.sica

Description Fonctionnelle

Objectif : Mettre à jour l'état des chèques dans la base de données, en particulier pour les chèques rejetés en raison de l'absence d'image.

Traitement :

Met à jour l'état des chèques dont l'état initial correspond à un état de rejet.

Assigne un motif de rejet spécifique aux chèques concernés.

Méthodes

OutgoUpdaterWriter() : Constructeur par défaut.

execute() : Exécute le processus de mise à jour des chèques dans la base de données.

Paramètres Utilisés

CETAOPEALLICOM2 : État final des chèques après mise à jour.

CETAOPERET : État initial des chèques à mettre à jour.

motifrejet : Motif de rejet assigné aux chèques mis à jour.

Remarques

Gestion des Erreurs : Les erreurs lors de la mise à jour des chèques sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de mises à jour ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et effectuer les mises à jour nécessaires.
