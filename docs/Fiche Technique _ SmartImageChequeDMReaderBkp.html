Introduction

Description : La tache SmartImageChequeDMReaderBkp est une extension de FlatFileReader utilisée pour lire et traiter les fichiers d'images de chèques scannés. Elle gère la validation des données, le traitement des images, et l'intégration des informations dans le système bancaire.

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers d'images de chèques dans le répertoire configuré.

Traitement : Vérifie les données des fichiers scannés, traite les images des chèques, et met à jour les informations dans la base de données.

Résultat : Mise à jour des statuts des chèques et des séquences dans la base de données, avec les images des chèques déplacées vers le répertoire approprié.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers : Assure la lecture, la validation, et le déplacement des fichiers d'images de chèques.

Interaction avec la Base de Données : Met à jour les statuts des chèques et des séquences.

Traitement d'Images : Redimensionne et compresse les images des chèques.

Vérification de Conformité : Vérifie si les fichiers ont déjà été traités pour éviter les doublons.

Lecture de Codes DataMatrix : Utilise DataMatrixClient pour extraire et traiter les données des codes DataMatrix présents sur les images de chèques. Ces données sont utilisées pour enrichir les informations sur le chèque et vérifier son authenticité.

Détection des Coordonnées de la DataMatrix

Paramètres de Configuration : Les coordonnées et la taille du sous-image contenant le code DataMatrix sont définies par des paramètres configurables :

DT_SNIPPET_X, DT_SNIPPET_Y, DT_SNIPPET_W, DT_SNIPPET_H: Coordonnées et dimensions pour les images de taille standard.

DT_SNIPPET_X1, DT_SNIPPET_Y1, DT_SNIPPET_W1, DT_SNIPPET_H1: Coordonnées et dimensions pour les grandes images.

Extraction du Sous-Image : Ces paramètres sont utilisés pour extraire un sous-image de l'image complète du chèque, qui est ensuite traité pour lire le code DataMatrix.

Traitement des Exceptions : Si la lecture du code échoue, un service Python est appelé pour tenter de lire le code DataMatrix.

Paramètres du Composant

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées.

CODE_SICA2 et CODE_SICA3 : Paramètres pour la conversion des codes numériques en codes alphanumériques.

COMPRESSION_RATIO : Ratio de compression pour les images.
