Introduction

Description : La tache SmartImageEffetSynReader est une extension de FlatFileReader utilisée pour lire et traiter les fichiers d'images d'effets synchronisés. Elle gère la validation des données, le traitement des images, et l'intégration des informations dans le système bancaire.

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers d'images d'effets dans le répertoire configuré.

Traitement : Vérifie les données des fichiers scannés, traite les images des effets, et met à jour les informations dans la base de données.

Résultat : Mise à jour des statuts des effets et des séquences dans la base de données, avec les images des effets déplacées vers le répertoire approprié.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers : Assure la lecture, la validation, et le déplacement des fichiers d'images d'effets.

Interaction avec la Base de Données : Met à jour les statuts des effets et des séquences.

Traitement d'Images : Redimensionne et compresse les images des effets.

Vérification de Conformité : Vérifie si les fichiers ont déjà été traités pour éviter les doublons.

Lecture de Codes DataMatrix : Utilise DataMatrixClient pour extraire et traiter les données des codes DataMatrix présents sur les images d'effets. Ces données sont utilisées pour enrichir les informations sur l'effet et vérifier son authenticité.

Paramètres du Composant

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées.

ENDOS_SNIPPET_WIDTH : Largeur du sous-image pour lire le numéro d'endossement.

ENDOS_SNIPPET_HEIGTH : Hauteur du sous-image pour lire le numéro d'endossement.

COMPRESSION_RATIO : Ratio de compression pour les images JPEG.

DELETE_IMAGES_AFTER_READ : Indicateur pour supprimer les images après lecture.

CODE_SICA2 : Code SICA de type 2 pour la conversion des codes numériques en alphanumériques.

CODE_SICA3 : Code SICA de type 3 pour la conversion des codes numériques en alphanumériques.

STRONG_CHECK_CMC7 : Indicateur pour activer la vérification stricte de la ligne CMC7.
