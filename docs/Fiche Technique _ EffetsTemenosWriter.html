Introduction

Description : La tache EffetsTemenosWriter est une extension de FlatFileWriter utilisée pour gérer l'envoi des effets vers le système Temenos. Elle traite les lots d'effets, en les préparant pour l'envoi sous forme de fichiers plats.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Préparer et envoyer des lots d'effets vers le système Temenos.

Traitement :

Récupère les effets depuis la base de données.

Génère un fichier plat contenant les informations des effets.

Met à jour l'état des effets et des remises après traitement.

Méthodes

EffetsTemenosWriter() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de préparation et d'envoi des effets.

prepareEffets(List<Cheques> cheques) : Prépare les effets pour l'envoi.

printLotCheque(List<Cheques> cheques, EnteteLot enteteLot) : Génère le fichier plat pour un lot d'effets.

printCheques(EnteteLot enteteLot, int j) : Génère la ligne de fichier pour un effet.

printEnteteLot(EnteteLot enteteLot) : Génère l'en-tête du fichier pour un lot d'effets.

Paramètres Utilisés

CETAOPEVAL : Paramètre indiquant l'état des opérations validées.

CETAOPEVAL2 : Paramètre indiquant un autre état des opérations validées.

CETAOPESAI : Paramètre indiquant l'état des opérations saisies.

CETAOPEALLICOM1 : Paramètre indiquant l'état des opérations prêtes pour l'envoi.

CETAOPEVALSURCAIENVSIB : Paramètre indiquant l'état des opérations envoyées au SIB.

Remarques

Gestion des Erreurs : Les erreurs lors de la préparation des effets sont gérées par des logs.

Extensibilité : La classe peut être étendue pour inclure d'autres types d'effets ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
