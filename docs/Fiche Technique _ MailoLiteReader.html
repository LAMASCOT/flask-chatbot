Introduction

Description : La tache MailoLiteReader est conçue pour lire et traiter les fichiers binaires contenant des informations sur les chèques et effets pour les clients corporatifs, en mettant à jour les enregistrements dans la base de données et en intégrant les images associées dans le système bancaire.

Package

Emplacement : sas.action.readers.corporates

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers binaires dans le répertoire configuré.

Traitement :

Lit le fichier pour extraire les en-têtes et les index d'images.

Met à jour ou insère les enregistrements de chèques et d'effets dans la base de données.

Gère les images associées aux chèques et effets, en les stockant dans le répertoire configuré.

Résultat : Les enregistrements de chèques et d'effets sont mis à jour dans la base de données, et les images sont stockées.

En cas d’erreur : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise des flux d'entrée pour lire les en-têtes et les index d'images des fichiers binaires.

Mise à Jour des Enregistrements :

Utilise DataBaseJPA pour exécuter des requêtes SQL qui mettent à jour ou insèrent les enregistrements de chèques et d'effets dans la base de données.

Gestion des Images :

Lit et stocke les images recto et verso des chèques et effets dans le répertoire configuré.

Paramètres du Composant

`MAILO_EXT_FOLDER` : Répertoire où les images des chèques et effets sont stockées.

`DELETE_MAILO_CORP` : Indique si le fichier source doit être supprimé après traitement.

Interactions avec la Base de Données

Requêtes SQL : Utilise des requêtes SQL pour mettre à jour ou insérer les enregistrements de chèques et d'effets dans la base de données.

Gestion des Transactions : Utilise DataBaseJPA pour gérer les transactions de base de données.

Paramètres Utilisés et Valeurs Possibles

`CETAOPERET` :

Utilisation : Code d'état pour les opérations en attente de traitement.

`CETAOPERETREC` :

Utilisation : Code d'état pour les opérations traitées avec succès.

`CETAOPERETIMA` :

Utilisation : Code d'état pour les opérations avec image associée.

`DELETE_MAILO_CORP` :

Utilisation : Indique si le fichier source doit être supprimé après traitement.

Valeurs Possibles : "OUI" pour activé, autre pour désactivé.
