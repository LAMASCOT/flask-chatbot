Introduction

Description : La tache ComptesBorneWriter est une extension de FlatFileWriter utilisée pour envoyer des informations de comptes vers des bornes. Elle récupère les comptes depuis la base de données et génère un fichier plat contenant les détails des comptes, qui est ensuite envoyé vers les bornes.

Package

Emplacement : sas.action.writers.borne

Description Fonctionnelle

Objectif : Générer et envoyer un fichier contenant les informations des comptes vers les bornes.

Traitement :

Récupère les comptes depuis la base de données.

Récupère les répertoires de destination pour les fichiers de comptes.

Génère un fichier plat pour chaque répertoire de destination contenant les informations des comptes.

Écrit les informations des comptes dans le fichier généré.

Attributs

db : DataBaseJPA - Instance pour l'accès à la base de données.

parameters : HashMap<String, Object> - Paramètres pour les requêtes SQL.

Méthodes

ComptesBorneWriter() : Constructeur par défaut qui initialise la description du processus.

execute() : Exécute le processus de génération et d'envoi des fichiers de comptes vers les bornes.

Paramètres Utilisés

NOMDOSCPTBOR : Nom du dossier de destination pour les fichiers de comptes.

NOMFICCPTBOR : Nom du fichier de comptes à générer.

SEPFICCPTBOR : Séparateur utilisé dans le fichier de comptes.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs.

Extensibilité : La classe est conçue pour être extensible, permettant l'ajout de nouvelles fonctionnalités ou l'intégration avec d'autres systèmes de bornes.

Utilisation de la Base de Données : Utilise DataBaseJPA pour gérer les connexions et les opérations sur la base de données.
