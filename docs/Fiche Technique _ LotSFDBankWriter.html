Introduction

Description : La tache LotSFDBankWriter est une extension de FlatFileWriter utilisée pour coordonner la génération de fichiers de lot d'opérations vers la banque. Elle utilise deux sous-écrivains, LotSFDBankNatWriter et LotSFDBankSrgWriter, pour gérer respectivement les opérations nationales et sous-régionales.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Coordonner la génération des fichiers de lot pour les opérations nationales et sous-régionales et les envoyer à la banque.

Traitement :

Initialise les sous-écrivains pour les opérations nationales et sous-régionales.

Exécute chaque sous-écrivain pour générer les fichiers de lot correspondants.

Met à jour la description du processus avec les résultats des sous-écrivains.

Attributs

lotNatWriter : LotSFDBankNatWriter - Sous-écrivain pour les opérations nationales.

lotSrgWriter : LotSFDBankSrgWriter - Sous-écrivain pour les opérations sous-régionales.

Méthodes

LotSFDBankWriter() : Constructeur par défaut qui initialise les sous-écrivains et la description du processus.

execute() : Exécute le processus de génération des fichiers de lot en utilisant les sous-écrivains pour les opérations nationales et sous-régionales.

Paramètres Utilisés

SIB_IN_FOLDER : Répertoire de destination pour les fichiers générés. Ce paramètre est utilisé par les sous-écrivains pour déterminer où enregistrer les fichiers de lot générés.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de transactions ou formats de fichiers.

Coordination : La classe LotSFDBankWriter coordonne la génération des fichiers de lot pour les transactions nationales et sous-régionales, assurant une gestion efficace des transferts pour les banques.
