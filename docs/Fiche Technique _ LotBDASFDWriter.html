Introduction

Description : La tache LotBDASFDWriter est une extension de FlatFileWriter utilisée pour générer des fichiers de lot d'opérations de prélèvements pour les banques. Elle traite spécifiquement les prélèvements et génère des fichiers de lot pour les envoyer aux banques.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Générer et envoyer des fichiers de lot de prélèvements à la banque.

Traitement :

Récupère les prélèvements à partir de la base de données.

Prépare les entêtes et les opérations pour les prélèvements.

Génère des fichiers de lot pour les prélèvements.

Met à jour l'état des prélèvements dans la base de données après traitement.

Attributs

sql : String - Requête SQL utilisée pour récupérer les données.

refLot : int - Référence du lot en cours de traitement.

vEnteteLots : Vector<EnteteLot> - Liste des entêtes de lot préparées.

prelevements25 : List<Prelevements> - Liste de prélèvements de type 25.

remcom : Remcom - Objet représentant une remise commerciale.

lotcom : Lotcom - Objet représentant un lot commercial.

lotFile : String - Nom du fichier de lot généré.

remiseHasPrelevements : boolean - Indicateur de présence de prélèvements dans la remise.

dataBaseJPA : DataBaseJPA - Instance pour l'accès à la base de données.

parameters : ConcurrentHashMap<String, Object> - Paramètres pour les requêtes SQL.

Méthodes

LotBDASFDWriter() : Constructeur par défaut qui initialise la description du processus.

execute() : Exécute le processus de génération des fichiers de lot pour les prélèvements.

returnEtabsUser(BigDecimal etat) : Récupère les établissements utilisateurs pour un état donné.

preparePrelevements(List<Prelevements> prelevements, String user, String codeEmetteur) : Prépare les prélèvements pour l'inclusion dans un fichier de lot.

feedOperationWithPrelevement25(Prelevements prelevement, int j, Operation[] operations) : Alimente une opération avec les détails d'un prélèvement.

fileNameNat(Object obj, EnteteLot enteteLot, String user) : Génère le nom de fichier pour un lot national.

fileNameNat(String codeEmetteur) : Génère le nom de fichier pour un lot de prélèvements.

printLotPrelevement(List<Prelevements> prelevements, EnteteLot enteteLot, String codeEmetteur) : Imprime le lot de prélèvements dans un fichier.

updateEtatOperations() : Met à jour l'état des opérations dans la base de données après traitement.

addErrorMessage(String message) : Ajoute un message d'erreur à l'interface utilisateur.

addWarnMessage(String message) : Ajoute un message d'avertissement à l'interface utilisateur.

addInfoMessage(String message) : Ajoute un message d'information à l'interface utilisateur.

Paramètres Utilisés

CETAOPEVALDELTA : Utilisé pour filtrer les opérations à traiter.

CETAOPEALLICOM1 : Utilisé pour mettre à jour l'état des opérations après traitement.

SIB_IN_FOLDER : Répertoire de destination pour les fichiers générés.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de transactions ou formats de fichiers.

Coordination : La classe LotBDASFDWriter coordonne la génération des fichiers de lot pour les prélèvements, assurant une gestion efficace des transferts pour les banques.
