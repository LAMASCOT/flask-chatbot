SmartEffetsRemManSynReader

Introduction

Description : La tache SmartEffetsRemManSynReader est conçue pour lire, traiter et synchroniser les fichiers d'effets scannés avec des informations de remise. Elle intègre les informations des effets dans le système, applique des vérifications de conformité, et gère le déplacement des fichiers d'images vers les répertoires appropriés.

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers d'effets scannés dans le répertoire configuré.

Traitement : Vérifie les données des fichiers scannés, traite les images des effets, et met à jour les informations dans la base de données.

Résultat : Mise à jour des statuts des effets et des séquences dans la base de données, avec les images des effets déplacées vers le répertoire approprié.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers : Assure la lecture, la validation, et le déplacement des fichiers d'effets scannés.

Interaction avec la Base de Données : Met à jour les statuts des effets et des séquences.

Traitement d'Images : Déplace et renomme les images des effets.

Vérification de Conformité : Vérifie si les fichiers ont déjà été traités pour éviter les doublons.

Paramètres du Composant

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées.

CODE_SICA2 et CODE_SICA3 : Paramètres pour la conversion des codes numériques en codes alphanumériques.

RVDSTRONGCHECK : Paramètre pour la gestion des fichiers déjà traités.

Exemples d'Utilisation

Utilisée pour valider, traiter et sauvegarder les effets scannés avec des informations de remise dans un environnement bancaire.

Intégration avec d'autres Systèmes

Intégrée avec le système de gestion des effets et les services de gestion des fichiers pour le traitement et le stockage des images.
