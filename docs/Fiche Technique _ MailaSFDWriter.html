Introduction

Description : La tache MailaSFDWriter est une extension de FlatFileWriter utilisée pour générer et envoyer des fichiers de type MAILA vers un partenaire. Elle crée un fichier plat contenant les détails de l'en-tête de remise et gère la configuration du nom de fichier.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Générer des fichiers MAILA pour l'envoi vers un partenaire.

Traitement :

Configure le nom du fichier MAILA.

Vérifie et crée le répertoire de destination si nécessaire.

Génère un en-tête de remise avec les informations nécessaires.

Crée un fichier plat avec l'en-tête de remise et une ligne de fin de fichier.

Méthodes

MailaSFDWriter() : Constructeur par défaut qui initialise la description de l'envoi MAILA.

MailaSFDWriter(String name) : Constructeur qui configure le nom du fichier MAILA en fonction du nom fourni.

execute() : Exécute le processus de génération du fichier MAILA, en créant l'en-tête de remise et en écrivant le fichier plat.

getNomFichierMaila() : Retourne le nom du fichier MAILA.

setNomFichierMaila(String nomFichierMaila) : Définit le nom du fichier MAILA.

Paramètres Utilisés

VERSION_SICA : Version du système SICA utilisée pour déterminer le format de l'en-tête de remise.

MAILA_SFD_NAT_FOLDER : Répertoire de destination pour les fichiers MAILA SFD nationaux.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération du fichier sont gérées par la classe parente FlatFileWriter. Si le répertoire de destination ne peut pas être créé, un événement d'erreur est enregistré.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de vérifications ou de formats de fichiers si nécessaire.

Configuration : La classe utilise des composants configurables pour la génération des fichiers, permettant une adaptation facile aux besoins spécifiques du système.

Fichiers Plats : Les fichiers plats générés contiennent les détails de l'en-tête de remise et une ligne de fin de fichier, et sont enregistrés dans le répertoire spécifié par MAILA_SFD_NAT_FOLDER.
