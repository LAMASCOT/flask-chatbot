Introduction

Description : La tache ChequesExportXLSWriter est une extension de BinFileWriter utilisée pour exporter les chèques transmis à la banque dans un fichier Excel. Elle génère un fichier XLS contenant les détails des chèques pour une date de traitement donnée.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Exporter les chèques transmis à la banque dans un fichier Excel.

Traitement :

Récupère les informations des chèques à partir de la base de données pour une date de traitement spécifique.

Génère un fichier Excel contenant les détails des chèques, tels que le numéro de chèque, le compte tiré, le nom du tiré, la date d'émission, le montant, le point d'accès, et l'utilisateur.

Enregistre le fichier Excel dans un répertoire spécifié.

Méthodes

ChequesExportXLSWriter() : Constructeur qui initialise la description de l'exportation des chèques.

execute() : Exécute le processus d'exportation des chèques en générant un fichier Excel avec les détails des chèques pour la date de traitement spécifiée.

Composants Internes

DataBaseJPA : Utilisé pour interagir avec la base de données et récupérer les informations des chèques.

HSSFWorkbook : Utilisé pour créer et manipuler des fichiers Excel au format HSSF (Excel 97-2003).

Sheet : Représente une feuille dans le classeur Excel.

Row : Représente une ligne dans la feuille Excel.

Paramètres Utilisés

SIB_IN_FOLDER : Répertoire de destination pour le fichier Excel généré.

CHQ_OUT_FILE_ROOTNAME : Nom de base pour le fichier Excel généré.

CETAOPEALLICOM1 : État des chèques à inclure dans l'exportation.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération du fichier Excel sont gérées par la classe parente BinFileWriter.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de données si nécessaire.

Configuration : La classe utilise des composants configurables pour la génération du fichier Excel, permettant une adaptation facile aux besoins spécifiques du système.

Coordination : La classe ChequesExportXLSWriter coordonne l'exportation des chèques en générant un fichier Excel détaillé, assurant une gestion efficace des chèques transmis à la banque.
