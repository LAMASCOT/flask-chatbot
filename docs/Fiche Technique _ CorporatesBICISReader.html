Introduction

Description : La tache CorporatesBICISReader est conçue pour lire et traiter les fichiers plats contenant des informations sur les remises et les chèques des clients corporatifs, en intégrant ces données dans la base de données et en gérant le stockage des images associées.

Package

Emplacement : sas.action.readers.corporates

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de remises dans le répertoire configuré.

Traitement :

Lit chaque ligne du fichier pour extraire les informations des remises et des chèques.

Crée des objets Remises et Cheques à partir des données extraites.

Stocke les images des chèques dans le répertoire configuré.

Insère les remises et les chèques dans la base de données.

Résultat : Les remises et les chèques sont intégrés et mis à jour dans la base de données, et les images sont stockées dans le répertoire approprié.

En cas d’erreur : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise un BufferedReader pour lire chaque ligne du fichier et extraire les informations nécessaires.

Création d'Objets :

Utilise la méthode getChamp pour extraire les champs des lignes et créer des objets Remises et Cheques.

Stockage des Images :

Utilise FileUtils.copyFile pour copier les images des chèques dans le répertoire configuré.

Mise à Jour des Enregistrements :

Utilise DataBaseJPA pour insérer les remises et les chèques dans la base de données.

Paramètres du Composant

`IMG_ALLER_FOLDER` : Dossier de destination pour les images traitées.

`IMG_ALLER_CORP` : Dossier source pour les images des chèques.

`VALIDATION_CORP_AUTO` : Indique si la validation automatique des remises est activée.

Interactions avec la Base de Données

Requêtes SQL : Utilise des requêtes SQL pour insérer les remises et les chèques dans la base de données.

Gestion des Transactions : Utilise DataBaseJPA pour gérer les transactions de base de données.

Paramètres Utilisés et Valeurs Possibles

`CETAOPEVAL` :

Utilisation : Code d'état pour les opérations en attente de validation.

Valeurs Possibles : Identifiants numériques définis dans le système.

`CETAOPESUPVAL` :

Utilisation : Code d'état pour les opérations validées automatiquement.

Valeurs Possibles : Identifiants numériques définis dans le système.
