   Introduction

Cette documentation présente les différents paramétrages de déploiement des applications sur l’application SMARTACCESS. Elle décrit les paramétrages à mettre en place.

L’objectif est d’offrir une vue d’ensemble sur les paramètres disponibles, notamment les bases de données, les serveurs d’application et les mécanismes d’interconnexion, afin de reproduire les sites déjà en production sur les environnements de test.

Cette documentation s’adresse aux équipes techniques impliquées dans le développement, l’intégration et l’exploitation des applications sur la plateforme SMARTAPS.
