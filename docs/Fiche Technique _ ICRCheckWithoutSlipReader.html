ICRCheckWithoutSlipReader

Introduction

Description : La tache ICRCheckWithoutSlipReader est conçue pour lire et traiter les fichiers de chèques scannés sans bordereau, en vérifiant l'intégrité des données et en intégrant les informations dans le système.

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de chèques scannés dans le répertoire configuré.

Traitement : Vérifie les données des fichiers scannés, traite les images des chèques, et met à jour les informations dans la base de données.

Résultat : Mise à jour des statuts des chèques et des séquences dans la base de données.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Gestion des Fichiers : Assure la lecture, la validation, et le déplacement des fichiers de chèques scannés.

Interaction avec la Base de Données : Met à jour les statuts des chèques et des séquences.

Traitement d'Images : Redimensionne et tatoue les images des chèques.

Paramètres du Composant

LINE_SCORE_PARAM : Paramètre de score de ligne utilisé pour la validation.

CORE_SYSTEM : Indique le système central utilisé (ex. BAOBAB).

RESIZE_THE_IMAGE : Indique si les images doivent être redimensionnées.

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées.

SAISIE_LENTE_DEV : Paramètre de développement pour la saisie lente.

LENTEUR_TIME_DEV : Temps de latence pour le développement.

Exemples d'Utilisation

Utilisée pour valider et traiter les chèques scannés sans bordereau dans un environnement bancaire.

Intégration avec d'autres Systèmes

Intégrée avec le système de gestion des chèques et le service ICR pour la reconnaissance des caractères.
