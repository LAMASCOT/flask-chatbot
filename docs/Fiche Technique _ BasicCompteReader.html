Introduction

Description : La tache BasicCompteReader est utilisée pour lire et traiter les fichiers contenant des informations de comptes bancaires, en effectuant des opérations de création, de mise à jour ou de suppression sur ces comptes dans la base de données.

Package

Emplacement : sas.action.readers.comptes

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de comptes dans le répertoire configuré.

Traitement :

Lit chaque ligne du fichier pour extraire les informations de comptes bancaires.

Effectue des opérations de création, mise à jour ou suppression sur les comptes en fonction des actions spécifiées dans le fichier.

Utilise des transactions en lot pour insérer ou mettre à jour les comptes dans la base de données.

Résultat : Les comptes bancaires sont créés, mis à jour ou supprimés dans la base de données selon les instructions du fichier.

En cas d’erreur : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise un BufferedReader pour lire chaque ligne du fichier et extraire les informations nécessaires.

Gestion des Comptes :

Crée de nouveaux comptes si l'action est "C".

Met à jour les comptes existants si l'action est "M".

Supprime les comptes si l'action est "A".

Interaction avec la Base de Données :

Utilise DataBaseJPA pour exécuter des requêtes SQL et gérer les transactions de base de données.

Utilise des transactions en lot pour optimiser l'insertion et la mise à jour des comptes.

Paramètres du Composant

Tatouage de la Date de Processus : La classe est configurée pour tatouer la date de traitement sur les fichiers.

Cette fiche technique inclut les fonctionnalités principales de la classe BasicCompteReader, en mettant l'accent sur le processus de lecture, de gestion des comptes et d'interaction avec la base de données.
