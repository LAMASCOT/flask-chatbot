Problème

Après installation d’une licence key et d’un UNLOCK KEY de scanner sur une machine, il peut arriver que cette licence “saute“ quelques jours plus tard et qu’un nouveau UNLOCK KEY est redemandée.

 Solution

Il existe 2 solutions pour ce type de problème:

  1. Générer un nouveau UNLOCK KEY à partir des informations affichées en se référant à la page Régénération d'une licence pour scanner

  2. Se procurer une nouvelle licence key et un nouveau Unlock key pour remplacer la licence key enregistrée dans la base de registre. Pour cette seconde option, ci-dessous la procédure à suivre:

    1. Ouvrir Regedit et Aller dans Ordinateur\HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Silver Bullet Technology\Installed Products\Ranger for Plug-ins\<modèle de scanner>\ClassA

    2. Changer la valeur ProductKey avec la nouvelle valeur de la licence key en votre possession.

    3. Revenir sur l'écran de RangerFlex en mode administrateur et saisir dans UNLOCK KEY, le nouveau unlock key généré.

    4. Valider et fermer la fenêtre.

Pour les agents de SmartAPS:

Il ne faut pas oublier de tester le scan depuis SmartACCESS afin de vérifier que tout fonctionne correctement.
