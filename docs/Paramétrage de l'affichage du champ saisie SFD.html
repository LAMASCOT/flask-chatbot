PARAMETRAGE DE L’AFFICHAGE DU CHAMP SAISIE SFD

Introduction

Ce paramétrage permet de contrôler l’affichage du champ relatif à la saisie des informations des SFD (Systèmes Financiers Décentralisés) dans l’application. Il est particulièrement utile dans les contextes où certaines banques collaborent avec des institutions de microfinance ou d’autres structures financières décentralisées.
En fonction de la valeur définie, ce champ peut être activé ou désactivé dynamiquement sans modification du code source, offrant ainsi une plus grande flexibilité dans le déploiement de l’application selon les besoins institutionnels.

Affichage du champ SAISIE SFD

  • Nom du paramètre : SAISIE_SFD_ACTIVE

  • Valeur par défaut : OUI

  • Libellé : SAISIE SFD ACTIVE

  • Type : CODE_PARAMS

Description :
Ce paramètre contrôle la visibilité du champ de saisie des données SFD dans l’interface de l’application.

  • Si la valeur est OUI, le champ est affiché et disponible à l’utilisateur.

  • Si la valeur est NON, le champ est masqué et ne figure pas dans l’interface.

Ce mécanisme permet une adaptation rapide de l’interface selon les exigences de l’entité bancaire sans impacter les autres utilisateurs ou clients institutionnels.
