Introduction

Description : La tache SmartSFTPWithCertUploader est conçue pour gérer le transfert de fichiers via le protocole SFTP (Secure File Transfer Protocol) en utilisant une authentification par certificat. Elle permet de transférer des fichiers vers un serveur distant de manière sécurisée.

Description Fonctionnelle

Déclenchement : La méthode treatFile() est appelée pour initier le processus de transfert de fichiers.

Traitement : Transfère les fichiers vers un serveur SFTP en utilisant une clé privée pour l'authentification.

Résultat : Les fichiers sont transférés vers un répertoire distant spécifié.

En cas d’erreur : Les erreurs de connexion ou de transfert sont consignées dans les journaux.

Opérations Clés

Connexion SFTP avec Certificat : Établit une connexion sécurisée avec le serveur SFTP en utilisant une clé privée pour l'authentification.

Transfert de Fichiers : Transfère les fichiers vers le répertoire distant spécifié sur le serveur SFTP.

Archivage des Fichiers : Déplace les fichiers transférés vers un répertoire d'archives local pour un suivi ultérieur.

Gestion des Sessions : Gère les sessions SFTP, y compris la connexion et la déconnexion sécurisées.

Paramètres du Composant

SFTPHOST : Adresse du serveur SFTP, obtenue dynamiquement.

SFTPPORT : Port du serveur SFTP, avec comme valeur par défaut de 22.

SFTPUSER : Nom d'utilisateur pour la connexion SFTP, obtenue dynamiquement.

SFTPPASSWORD : Mot de passe pour la connexion SFTP, qui peut être crypté, obtenue dynamiquement.

SFTPWORKINGDIR : Répertoire distant où les fichiers doivent être transférés, avec comme valeur par défaut /var/etc/SFTPWORKINGDIR.

SSH_KEY : Chemin vers la clé privée utilisée pour l'authentification SFTP avec comme valeur par défaut /etc/ssh/private_key.pem.

CRYPT_TYPE : Type de cryptage utilisé pour le mot de passe, si applicable avec comme valeur par défaut AES_256.

Exemples d'Utilisation

Utilisée pour automatiser le transfert sécurisé de fichiers vers un serveur SFTP dans un environnement d'entreprise, en assurant que les fichiers sont correctement transférés et archivés.

Intégration avec d'autres Systèmes

Intégrée avec des systèmes de gestion de fichiers et des bases de données pour assurer un traitement fiable et efficace des fichiers transférés.
