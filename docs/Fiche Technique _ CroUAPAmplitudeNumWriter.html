Introduction

Description : La tache CroUAPAmplitudeNumWriter est une extension de FlatFileWriter utilisée pour gérer l'envoi de lots d'opérations de retour vers le SIB. Elle traite les chèques, virements, et effets, en les préparant pour l'envoi sous forme de fichiers plats.

Package

Emplacement : sas.action.writers.uap

Description Fonctionnelle

Objectif : Préparer et envoyer des lots d'opérations de retour vers le SIB.

Traitement :

Récupère les chèques, virements, et effets depuis la base de données.

Génère des fichiers plats contenant les informations des transactions.

Met à jour l'état des transactions après traitement.

Méthodes

CroUAPAmplitudeNumWriter() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de préparation et d'envoi des transactions.

prepareCheques(List<Cheques> cheques) : Prépare les chèques pour l'envoi.

prepareVirements(List<Virements> virements) : Prépare les virements pour l'envoi.

prepareEffets(List<Effets> effets) : Prépare les effets pour l'envoi.

prepareRejetCheques(List<Cheques> cheques) : Prépare les rejets de chèques.

prepareRejetEffets(List<Effets> effets) : Prépare les rejets d'effets.

Paramètres Utilisés

CETAOPERETREC : Paramètre indiquant l'état des opérations reçues.

CETAOPERET : Paramètre indiquant l'état des opérations de retour.

CETAOPEREJRET : Paramètre indiquant l'état des opérations rejetées.

Remarques

Gestion des Erreurs : Les erreurs lors de la préparation des transactions sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe peut être étendue pour inclure d'autres types de transactions ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
