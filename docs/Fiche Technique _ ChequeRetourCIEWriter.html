Introduction

Description : La tache ChequeRetourCIEWriter est une extension de FlatFileWriter utilisée pour envoyer des chèques "Retour" vers le Système d'Information Bancaire (SIB). Elle traite les chèques en fonction de leur état et les insère dans les tables appropriées pour la compensation et le retour.

Package

Emplacement : sas.action.writers.cie

Description Fonctionnelle

Objectif : Envoyer les chèques "Retour" vers le SIB pour traitement.

Traitement :

Récupère les chèques avec des états spécifiques depuis la base de données.

Insère les chèques dans les tables IBUS_CHQ_RET et IBUS_CHQ_COMP selon leur type (Retour, Aller accepté, Aller rejeté).

Met à jour l'état des chèques après insertion réussie.

Attributs

db : DataBase - Instance pour l'accès à la base de données.

Méthodes

ChequeRetourCIEWriter() : Constructeur par défaut qui initialise la description du processus.

execute() : Exécute le processus d'envoi des chèques "Retour" vers le SIB.

Paramètres Utilisés

CETAOPERETREC, CETAOPERET : États des chèques à récupérer pour le retour.

CETAOPERETENVSIB, CETAOPERETRECENVSIB : Nouveaux états à attribuer aux chèques après envoi réussi.

CETAOPEALLICOM1ACCENVSIB, CETAOPEALLICOM1ACCENVSIBCON : États pour les chèques "Aller" acceptés.

CETAOPEREJRET, CETAOPEREJRETENVSIB : États pour les chèques "Aller" rejetés.

Remarques

Gestion des Erreurs : Les erreurs lors de l'insertion des chèques sont gérées par des logs, et un message d'avertissement est enregistré en cas d'échec.

Extensibilité : La classe est conçue pour être extensible, permettant l'ajout de nouvelles fonctionnalités ou l'intégration avec d'autres systèmes de compensation.

Utilisation de la Base de Données : Utilise DataBase et JDBCXmlReader pour gérer les connexions et les opérations sur la base de données.
