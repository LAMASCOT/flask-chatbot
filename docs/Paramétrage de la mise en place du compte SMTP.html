PARAMETRAGE DE LA MISE EN PLACE DU COMPTE SMTP

Pour le paramétrage du compte SMTP, il faut se rendre dans Payara/resources/JavaMail Sessions

 

Créer un JNDI Name comme indiqué ci-dessous :

Ensuite aller dans Additionnal Properties et ajouter les propriètés suivantes :

  1. EMAIL_WEBSERVICE_BASE_URI

    • Valeur : http://localhost:35924/rest

    • Libellé : EMAIL WEB SERVICE URI

    • Type : CODE_PARAMS

    • Description : Ce paramètre représente l’URL de base du service web utilisé pour l’envoi d’emails. Il est utilisé pour construire les appels HTTP vers le service de messagerie.

  2. SEND_EMAIL_ICOMA

    • Valeur : OUI

    • Libellé : SEND_EMAIL_ICOMA

    • Type : CODE_PARAMS

    • Description : Ce paramètre active ou désactive l'envoi d'emails via le service nommé "ICOMA". La valeur OUI indique que la fonctionnalité est activée.
