Introduction

Description : La tache DeprecatedMailiIntReader est conçue pour lire et traiter les fichiers binaires contenant des informations sur les remises de chèques et d'effets, en intégrant ces données dans la base de données et en gérant le stockage des images associées.

Package

Emplacement : sas.action.readers.corporates

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de remises dans le répertoire configuré.

Traitement :

Lit les en-têtes et les index d'images des fichiers binaires.

Insère les informations des remises et des chèques/effets dans la base de données.

Stocke les images des chèques/effets dans le répertoire configuré.

Gère les doublons de chèques en les enregistrant dans une table dédiée.

Résultat : Les remises et les chèques/effets sont intégrés et mis à jour dans la base de données, et les images sont stockées dans le répertoire approprié.

En cas d’erreur : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise BufferedInputStream pour lire les fichiers binaires.

Création d'Objets :

Utilise des méthodes pour extraire les champs des en-têtes et des index d'images et créer des objets Remises, Cheques, et Effets.

Stockage des Images :

Utilise BufferedOutputStream et ImageIO pour écrire les images des chèques/effets dans le répertoire configuré.

Mise à Jour des Enregistrements :

Utilise DataBaseJPA pour insérer les remises et les chèques/effets dans la base de données.

Gère les doublons de chèques en les enregistrant dans une table dédiée.

Paramètres du Composant

`IMG_ALLER_FOLDER` : Dossier de destination pour les images traitées.

`CORPORATE_OFFLINE` : Indique si le mode hors ligne est activé pour les corporates.

`TATOO_PICTURE` : Indique si les images doivent être tatouées avec des informations supplémentaires.

Interactions avec la Base de Données

Requêtes SQL : Utilise des requêtes SQL pour insérer et mettre à jour les remises et les chèques/effets dans la base de données.

Gestion des Transactions : Utilise DataBaseJPA pour gérer les transactions de base de données.

Paramètres Utilisés et Valeurs Possibles

`CETAOPEVAL` :

Utilisation : Code d'état pour les opérations en attente de validation.

Valeurs Possibles : Identifiants numériques définis dans le système.

`CETAOPEVAL2` :

Utilisation : Code d'état pour les opérations validées.

Valeurs Possibles : Identifiants numériques définis dans le système.
