Introduction

Description : La tache SmartSFTPDownloader est responsable du téléchargement de fichiers à partir d'un serveur SFTP. Elle gère la connexion, le téléchargement, et la suppression des fichiers distants après leur transfert.

Description Fonctionnelle

Déclenchement : La méthode execute() est appelée pour initier le processus de téléchargement.

Traitement : Établit une connexion SFTP, télécharge les fichiers du répertoire spécifié, et les enregistre localement. Les fichiers téléchargés sont ensuite supprimés du serveur SFTP.

Résultat : Les fichiers sont téléchargés avec succès et enregistrés dans le répertoire local spécifié.

En cas d’erreur : Les erreurs de connexion ou de transfert sont consignées dans les journaux, et des messages d'erreur sont affichés.

Opérations Clés

Connexion SFTP : Établit une connexion sécurisée au serveur SFTP en utilisant un mot de passe ou une clé privée.

Téléchargement de Fichiers : Télécharge les fichiers du répertoire distant spécifié et les enregistre localement.

Suppression de Fichiers : Supprime les fichiers du serveur SFTP après leur téléchargement réussi.

Gestion des Erreurs : Gère les erreurs de connexion et de transfert, en les consignant dans les journaux.

Paramètres du Composant

SFTPHOST : Adresse du serveur SFTP, obtenue dynamiquement à partir de l'identifiant du répertoire.

SFTPPORT : Port du serveur SFTP, avec une valeur par défaut de 22.

SFTPUSER : Nom d'utilisateur pour la connexion SFTP, obtenu dynamiquement.

SFTPPASS : Mot de passe pour la connexion SFTP, obtenu dynamiquement.

privateKeyPath : Chemin vers la clé privée pour l'authentification par clé, avec une valeur par défaut etc/ssh/id.

TIME_ALIVE_SFTP : Intervalle de temps pour garder la connexion SFTP active, avec une valeur par défaut de 300000ms.

preferredAuthentications : Méthode d'authentification préférée (mot de passe ou clé), avec une valeur par défaut password.

SFTPHEARTBEAT : Intervalle de temps en millisecondes entre les cycles de téléchargement pour permettre à la connexion de rester active, avec une valeur par défaut de 500000ms.

Exemples d'Utilisation

Utilisée pour automatiser le téléchargement de fichiers à partir de serveurs SFTP dans un environnement d'entreprise.

Intégration avec d'autres Systèmes

Intégrée avec les systèmes de gestion de fichiers pour assurer un transfert sécurisé et fiable des fichiers.
