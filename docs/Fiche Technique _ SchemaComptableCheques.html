Introduction

Description : La tache SchemaComptableCheques est utilisée pour gérer les comptes comptables associés aux opérations de chèques. Elle récupère les paramètres de compte à partir de la configuration et fournit des méthodes pour manipuler ces comptes dans le cadre des opérations de chèques.

Package

Emplacement : sas.action.writers.forimal

Description Fonctionnelle

Objectif : Gérer les comptes comptables pour les opérations de chèques, y compris l'encaissement, le recouvrement, et les frais associés.

Traitement :

Récupère les comptes comptables à partir des paramètres de configuration.

Fournit des méthodes pour créer et manipuler les entrées comptables liées aux chèques.

Méthodes

SchemaComptableCheques() : Constructeur par défaut.

createPortfeuilleCheque(Cheques c) : Crée une entrée comptable pour la mise en portefeuille d'un chèque.

Attributs

valeurEncaissementNonDisponible : Compte pour les valeurs à l'encaissement non disponibles.

cptClienteleExigibleEncaissement : Compte pour les comptes de la clientèle exigible après encaissement.

cptRecouvrement : Compte de recouvrement.

cptContrePartieClient : Compte de contrepartie pour le recouvrement client.

cptFraisBCEAO : Compte pour les frais BCEAO.

compteBCEAO : Compte pour le règlement BCEAO.

valeurImputees : Compte pour les valeurs à imputer.

compteLiaisonInterAgence : Compte de liaison inter-agence.

compteCommissionRejet : Compte pour les commissions de rejet.

compteTaxe : Compte pour la taxe (TAF).

montantCommissionRejet : Montant de la commission de rejet.

TAF : Taux de la taxe (TAF).

LOG : Logger pour enregistrer les événements et erreurs.

Paramètres Utilisés

CPT_ENCAISSEMENT_NON_DISPONIBLE, CPT_CLIENTELE_EXIGIBLE_ENCAISSEMENT, CPT_RECOUVREMENT, CPT_CONTREPARTIE_CLIENT, CPT_FRAIS_BCEAO, CPT_REGLEMENT_BCEAO, CPT_VALEUR_IMPUTEES, CPT_LIAISON_INTER_AGENCE, CPT_COMMISSION_REJET, CPT_TAXE, COMDEBREJCHQALLFONDIMAL, TAF : Paramètres de configuration pour les comptes comptables et les montants.

Remarques

Gestion des Erreurs : Utilise un logger pour enregistrer les erreurs et les événements importants.

Extensibilité : La tache peut être étendue pour inclure d'autres types de comptes ou opérations comptables.

Intégration : Conçue pour être utilisée dans des contextes où les opérations de chèques nécessitent une gestion comptable précise et conforme aux normes bancaires.
