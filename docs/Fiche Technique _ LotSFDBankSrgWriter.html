Introduction

Description : La tache LotSFDBankSrgWriter est une extension de FlatFileWriter utilisée pour générer des fichiers de lot d'opérations sous-régionales vers la banque. Elle traite différents types d'opérations financières, notamment les virements, les chèques, les effets, et les prélèvements.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Générer et envoyer des fichiers de lot d'opérations sous-régionales à la banque.

Traitement :

Récupère les opérations financières à partir de la base de données.

Prépare les entêtes et les opérations pour chaque type d'opération.

Génère des fichiers de lot pour chaque type d'opération.

Met à jour l'état des opérations dans la base de données après traitement.

Attributs

sql : String - Requête SQL utilisée pour récupérer les données.

refLot : int - Référence du lot en cours de traitement.

vEnteteLots : Vector<EnteteLot> - Liste des entêtes de lot préparées.

cheques : Cheques[] - Tableau de chèques à traiter.

cheques35 : List<Cheques> - Liste de chèques de type 35.

virements15 : List<Virements> - Liste de virements de type 15.

effets45 : List<Effets> - Liste d'effets de type 45.

effets46 : List<Effets> - Liste d'effets de type 46.

prelevements25 : List<Prelevements> - Liste de prélèvements de type 25.

remcom : Remcom - Objet représentant une remise commerciale.

lotcom : Lotcom - Objet représentant un lot commercial.

lotFile : String - Nom du fichier de lot généré.

remiseHasCheques : boolean - Indicateur de présence de chèques dans la remise.

remiseHasEffets : boolean - Indicateur de présence d'effets dans la remise.

remiseHasVirements : boolean - Indicateur de présence de virements dans la remise.

remiseHasPrelevements : boolean - Indicateur de présence de prélèvements dans la remise.

dataBaseJPA : DataBaseJPA - Instance pour l'accès à la base de données.

parameters : ConcurrentHashMap<String, Object> - Paramètres pour les requêtes SQL.

Méthodes

LotSFDBankSrgWriter() : Constructeur par défaut qui initialise la description du processus.

execute() : Exécute le processus de génération des fichiers de lot pour les opérations sous-régionales.

prepareCheques(List<Cheques> cheques, String user) : Prépare les chèques pour l'inclusion dans un fichier de lot.

preparePrelevements(List<Prelevements> prelevements, String user) : Prépare les prélèvements pour l'inclusion dans un fichier de lot.

prepareEffets(List<Effets> effets, String user) : Prépare les effets pour l'inclusion dans un fichier de lot.

prepareVirements(List<Virements> virements, String user) : Prépare les virements pour l'inclusion dans un fichier de lot.

updateEtatOperations() : Met à jour l'état des opérations dans la base de données après traitement.

Paramètres Utilisés

CETAOPEVALDELTA : Utilisé pour filtrer les opérations à traiter.

CETAOPELOTGEN : Utilisé pour mettre à jour l'état des opérations après traitement.

SIB_IN_FOLDER : Répertoire de destination pour les fichiers générés.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de transactions ou formats de fichiers.

Coordination : La classe LotSFDBankSrgWriter coordonne la génération des fichiers de lot pour les transactions sous-régionales, assurant une gestion efficace des transferts pour les banques.
