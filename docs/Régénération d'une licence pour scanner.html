Problème

Après installation d’une licence key et d’un UNLOCK KEY de scanner sur une machine, il peut arriver que cette licence “saute“ quelques jours plus tard et qu’un nouveau UNLOCK KEY est redemandée.

 Solution

Pour résoudre cet incident, il faut récupérer des infos dans la base de registre de cette machine en suivant les étapes ci-dessous:

  1. Ouvrez la boîte de dialogue Exécuter la commande en appuyant sur la touche Windows + R
    ou
    Recherchez "Exécuter" dans le menu Démarrer

  2. Dans la boîte de dialogue Exécuter la commande, tapez "regedit" et cliquez sur OK

  3. Windows peut vous demander d'autoriser l'éditeur du registre à apporter des modifications ; cliquez sur Oui

  4. Sur le côté gauche de l'écran, il y aura un menu "Arbre".

    Dans cet arbre, recherchez :
    Système d'exploitation 32 bits - HKEY_LOCAL_MACHINE\SOFTWARE\Silver Bullet Technology
    ou
    Système d'exploitation 64 bits - HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Silver Bullet Technology\

  5. Cliquez avec le bouton droit sur le dossier "Silver Bullet Technology"> Exporter

  6. Enregistrez l'exportation dans un fichier texte (exemple : Reg.txt)

  7. Transmettre ce fichier à SmartAPS qui vous reviendra dans la journée avec un nouveau UNLOCK KEY.

Pour les agents de SmartAPS:

Transmettre le fichier Reg.txt au Helpdesk sbullet (https://sbullet.com/Tickets ) , avec une capture écran du rangerFlex, qui libérera la licence pour permettre la génération d’un nouveau UNLOCK KEY.
