Introduction

Description : La tache ChequeSurCaisseSitWriter est une extension de FlatFileWriter utilisée pour envoyer les chèques sur caisses vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques sur caisses à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques sur caisses vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Vérifie si le verrouillage des chèques sur caisses est activé.

Récupère les chèques sur caisses à envoyer.

Génère un fichier plat avec les informations des chèques sur caisses.

Met à jour l'état des chèques et des remises après envoi.

Gère le verrouillage des chèques sur caisses pour éviter les conflits d'accès.

Méthodes

ChequeSurCaisseSitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des chèques sur caisses vers le SIB.

Paramètres Utilisés

LOT_CHEQUE_NC_ETAT_DEBUT : État initial des chèques sur caisses.

LOT_CHEQUE_NC_ETAT_FIN : État final des chèques sur caisses après envoi.

LOCK_CHK_NC_ALLER : Paramètre de verrouillage pour éviter les conflits d'accès lors de la composition des fichiers de chèques sur caisses.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques sur caisses sont gérées par des logs et des messages d'avertissement.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et effectuer les mises à jour nécessaires.
