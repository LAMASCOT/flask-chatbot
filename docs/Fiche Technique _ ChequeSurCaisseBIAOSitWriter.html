Introduction

Description : La tache ChequeSurCaisseBIAOSitWriter est une extension de FlatFileWriter utilisée pour envoyer les chèques sur caisses vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques sur caisses à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques sur caisses vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Met à jour l'état des chèques avec des montants manquants.

Récupère les chèques sur caisses à envoyer.

Génère un fichier plat avec les informations des chèques sur caisses.

Met à jour l'état des chèques et des remises après envoi.

Méthodes

ChequeSurCaisseBIAOSitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des chèques sur caisses vers le SIB.

Paramètres Utilisés

CETAOPEERR : État des chèques avec erreur de validation.

CETAOPEVAL : État des chèques prêts à être validés.

CETAOPEVALSURCAIENVSIB : État des chèques sur caisses après envoi.

CETAREMENVSIB : État des remises après envoi.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques sur caisses sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBase pour interagir avec la base de données et effectuer les mises à jour nécessaires.
