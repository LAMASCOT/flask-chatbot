Introduction

Description : La tache CraUAPWriter est une extension de FlatFileWriter utilisée pour générer et envoyer des comptes rendus d'acquisition vers le SIB Temenos. Elle traite différents types d'opérations bancaires, y compris les chèques, virements, effets, et prélèvements, en fonction de l'état des remises.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Générer des fichiers de comptes rendus d'acquisition pour différentes opérations bancaires et les envoyer au SIB Temenos.

Traitement :

Récupère les données des opérations bancaires à partir de la base de données.

Prépare les opérations en fonction de leur type et de leur état (acceptation, rejet, annulation).

Génère des fichiers de comptes rendus pour chaque type d'opération.

Gère les références et les identifiants pour chaque opération.

Méthodes

CraUAPWriter() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de génération et d'envoi des comptes rendus d'acquisition.

prepareRejetCheques(Cheques[] cheques) : Prépare les comptes rendus de rejet pour les chèques.

prepareRejetPrelevements(Prelevements[] prelevements) : Prépare les comptes rendus de rejet pour les prélèvements.

prepareRejetEffets(Effets[] effets) : Prépare les comptes rendus de rejet pour les effets.

feedOperationWith... : Plusieurs méthodes pour alimenter les objets Operation avec les détails spécifiques des chèques, virements, effets, et prélèvements.

Paramètres Utilisés

CONSTANTE_SICA : Constante utilisée pour déterminer la source de l'opération.

CETAOPEALLICOM1ACC, CETAOPEALLICOM1REJ, etc. : Paramètres utilisés pour déterminer l'état des opérations.

Remarques

Gestion des Erreurs : Les erreurs lors de la récupération ou du traitement des données sont gérées par des logs, et la description de l'opération est mise à jour pour refléter le statut de chaque étape.

Extensibilité : La classe peut être étendue pour inclure d'autres types d'opérations ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
