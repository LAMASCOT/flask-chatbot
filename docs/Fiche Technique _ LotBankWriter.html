Introduction

Description : La tache LotBankWriter est une extension de FlatFileWriter utilisée pour générer des fichiers de type .LOT destinés à la banque. Elle coordonne l'exécution de deux sous-écrivains, LotBankNatWriter et LotBankSrgWriter, pour produire les fichiers requis.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Générer des fichiers .LOT pour la banque en utilisant des sous-écrivains spécialisés.

Traitement :

Initialise les sous-écrivains pour les fichiers nationaux et Srg.

Exécute chaque sous-écrivain pour générer les fichiers correspondants.

Met à jour la description avec les résultats de chaque sous-écrivain.

Méthodes

LotBankWriter() : Constructeur qui initialise la description de la génération des fichiers .LOT et instancie les sous-écrivains LotBankNatWriter et LotBankSrgWriter.

execute() : Exécute le processus de génération des fichiers .LOT, en appelant les méthodes execute() des sous-écrivains et en combinant leurs descriptions.

Composants Internes

LotBankNatWriter : Sous-écrivain responsable de la génération des fichiers .LOT pour les opérations nationales.

LotBankSrgWriter : Sous-écrivain responsable de la génération des fichiers .LOT pour les opérations Srg.

Paramètres Utilisés

ParametersMap : Carte de paramètres utilisée pour configurer les sous-écrivains. Elle est transmise à chaque sous-écrivain pour s'assurer qu'ils disposent des informations nécessaires pour générer les fichiers correctement.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par les sous-écrivains respectifs.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de sous-écrivains si nécessaire.

Configuration : La classe utilise des composants configurables pour la génération des fichiers, permettant une adaptation facile aux besoins spécifiques du système.

Coordination : La classe LotBankWriter agit comme un coordinateur, orchestrant l'exécution des sous-écrivains et consolidant leurs descriptions pour fournir un aperçu complet du processus de génération des fichiers .LOT.
