Introduction

Description : La tache ChequeRetourBFASitWriter est une extension de FlatFileWriter utilisée pour envoyer les chèques de retour vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques de retour à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques de retour vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Récupère les chèques de retour à envoyer.

Génère un fichier plat avec les informations des chèques de retour.

Met à jour l'état des chèques après envoi.

Méthodes

ChequeRetourBFASitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des chèques de retour vers le SIB.

Paramètres Utilisés

CETAOPERETREC : État des chèques de retour reçus prêts à être envoyés.

CETAOPERET : État des chèques de retour prêts à être envoyés.

CETAOPERETENVSIB : État des chèques de retour après envoi.

CETAOPERETRECENVSIB : État des chèques de retour reçus après envoi.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques de retour sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBase pour interagir avec la base de données et effectuer les mises à jour nécessaires.
