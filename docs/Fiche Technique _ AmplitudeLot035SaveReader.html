Introduction

Description : La tache AmplitudeLot035SaveReader est utilisée pour lire et traiter les fichiers de lots d'amplitude, en particulier pour sauvegarder les informations de chèques et gérer les images associées.

Package

Emplacement : sas.action.readers.ampup

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de lots d'amplitude dans le répertoire configuré.

Traitement :

Lit les lignes du fichier pour extraire les informations de chèques, telles que les en-têtes de lot et les opérations.

Vérifie l'existence des chèques dans la base de données pour éviter les doublons.

Insère de nouveaux chèques et met à jour les informations existantes.

Gère les images associées aux chèques en les déplaçant vers le répertoire approprié.

Résultat : Les informations extraites sont insérées dans la base de données, mettant à jour les tables de chèques et de remises. Les images des chèques sont déplacées vers le répertoire approprié.

En cas d’erreur : Les erreurs sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Assure la lecture et le traitement des lignes du fichier de lots d'amplitude.

Interaction avec la Base de Données :

Insère et met à jour les informations de chèques et de remises dans la base de données.

Utilise ComptesClient pour récupérer des informations de compte.

Gestion des Images :

Déplace les images des chèques vers le répertoire approprié et met à jour les chemins d'accès dans la base de données.

Vérifie l'existence des fichiers d'image avant de les copier.

Vérification de Conformité : Vérifie si les chèques ont déjà été traités pour éviter les doublons.

Paramètres du Composant

FILE_PATH_IMG_JPG : Chemin d'accès pour les images JPEG des chèques, avec comme valeur par défaut: /data/amplitude/images_jpg/.

IMG_ALLER_FOLDER : Répertoire de destination pour les images traitées, avec comme valeur par défaut: /data/amplitude/images_traitees/.

DATE_FOLDER_IMG_JPG : Indicateur pour inclure la date dans le chemin d'accès des images. Valeurs possibles : "OUI" pour inclure la date, "NON" pour ne pas l'inclure.
