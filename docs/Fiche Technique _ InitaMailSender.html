Introduction

Description : La tache InitaMailSender est une extension de FlatFileWriter utilisée pour envoyer des emails d'initialisation de période de compensation. Elle utilise des modèles d'email pour formater le contenu et envoie les emails aux destinataires spécifiés.

Package

Emplacement : sas.action.writers.sica.mail

Description Fonctionnelle

Objectif : Envoyer des emails d'initialisation de période de compensation.

Traitement :

Vérifie si le système de base est "BAOBAB" pour déterminer le traitement spécifique à la BCEAO.

Utilise un modèle d'email pour formater le contenu de l'email.

Envoie l'email aux destinataires spécifiés.

Méthodes

InitaMailSender() : Constructeur par défaut.

execute() : Exécute le processus d'envoi d'email d'initialisation de période de compensation.

getTemplateINITA() : Retourne le chemin du modèle d'email INITA.

setTemplateINITA(String templateINITA) : Définit le chemin du modèle d'email INITA.

getEmailsDest() : Retourne la liste des destinataires des emails.

setEmailsDest(String emailsDest) : Définit la liste des destinataires des emails.

getDateCompensation() : Retourne la date de compensation.

setDateCompensation(String dateCompensation) : Définit la date de compensation.

Paramètres Utilisés

TEMPLATE_INITA : Chemin du modèle d'email pour l'initialisation.

EMAILS_DEST_OPS : Liste des destinataires des emails.

CORE_SYSTEM : Indicateur du système de base utilisé, pour déterminer le traitement spécifique.

codeBanque : Code de la banque utilisé pour récupérer les destinataires spécifiques.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des emails sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe peut être étendue pour inclure d'autres types de notifications ou de traitements spécifiques.

Intégration : Cette classe utilise JerseyEmailClient pour l'envoi des emails, facilitant ainsi l'intégration avec d'autres services de messagerie.
