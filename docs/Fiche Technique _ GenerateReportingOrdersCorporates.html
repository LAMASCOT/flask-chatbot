Introduction

Description : La tache GenerateReportingOrdersCorporates est utilisée pour générer des rapports sur les ordres de paiement pour les entreprises. Elle génère des rapports PDF pour les chèques et effets payés ou rejetés, et envoie ces rapports par email aux établissements concernés.

Package

Emplacement : sas.action.writers.corporates

Description Fonctionnelle

Objectif : Générer et envoyer des rapports sur les ordres de paiement pour les entreprises.

Traitement :

Récupère les informations des comptes d'entreprise à partir de la base de données.

Génère des rapports PDF pour les chèques et effets payés ou rejetés.

Envoie les rapports par email aux établissements concernés.

Méthodes

GenerateReportingOrdersCorporates() : Constructeur qui initialise les composants nécessaires pour la génération des rapports.

genChequesPayesReport(HashSet<String> comptesCorp, String date) : Génère et envoie un rapport pour les chèques payés.

genChequesRejReport(HashSet<String> comptesCorp, String date) : Génère et envoie un rapport pour les chèques rejetés.

genEffetsPayesReport(HashSet<String> comptesCorp, String date) : Génère et envoie un rapport pour les effets payés.

genEffetsRejReport(HashSet<String> comptesCorp, String date) : Génère et envoie un rapport pour les effets rejetés.

Composants Internes

DataBaseJPA : Utilisé pour interagir avec la base de données et récupérer les informations des comptes et établissements.

EmailRequest : Représente une requête d'envoi d'email avec un fichier joint.

JerseyEmailClient : Client utilisé pour envoyer des emails.

Paramètres Utilisés

EMAIL_TEMPLATES_FOLDER : Répertoire contenant les modèles d'email.

PatwareLibMvnPU : Unité de persistance utilisée pour générer les rapports PDF.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des rapports ou de l'envoi des emails sont gérées par des logs de débogage.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres types de rapports si nécessaire.

Configuration : La classe utilise des composants configurables pour la génération des rapports et l'envoi des emails, permettant une adaptation facile aux besoins spécifiques du système.

Coordination : La classe GenerateReportingOrdersCorporates coordonne la génération et l'envoi des rapports pour différents types d'opérations, assurant une gestion efficace des ordres de paiement pour les entreprises.
