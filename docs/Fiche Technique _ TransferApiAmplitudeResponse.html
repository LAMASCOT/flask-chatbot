Introduction

Description : La tache TransferApiAmplitudeResponse est utilisée pour représenter la réponse d'une API lors de l'envoi de virements vers le système Amplitude. Elle contient des informations sur l'opération de transfert, y compris l'identifiant, la référence de l'opération, le nombre de transactions, le montant total, la date de soumission, le statut, et les éventuelles erreurs.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Représenter la réponse d'une API pour les transferts de virements.

Traitement :

Stocke les informations renvoyées par l'API après l'envoi d'une requête de transfert.

Fournit des méthodes pour accéder et modifier ces informations.

Attributs

id : BigInteger - Identifiant unique de l'opération de transfert.

referenceOperation : String - Référence de l'opération de transfert.

debitIban : String - IBAN de débit utilisé pour le transfert.

nombreTransaction : int - Nombre total de transactions dans l'opération de transfert.

montantTotal : BigInteger - Montant total des transactions dans l'opération de transfert.

dateSoumission : String - Date et heure de soumission de l'opération de transfert, formatée selon le pattern yyyy-MM-dd'T'HH:mm:ss.SSSXXX.

statut : String - Statut actuel de l'opération de transfert.

error : GenericError - Objet contenant des informations sur les erreurs éventuelles.

Méthodes

TransferApiAmplitudeResponse() : Constructeur par défaut.

getters et setters : Méthodes pour accéder et modifier les attributs de la classe.

toString() : Retourne une représentation sous forme de chaîne de caractères de l'objet, incluant tous ses attributs.

Composants Internes

GenericError : Classe utilisée pour représenter les erreurs éventuelles dans la réponse de l'API.

Paramètres Utilisés

JsonFormat : Annotation utilisée pour spécifier le format de la date de soumission dans la réponse JSON.

Temporal : Annotation utilisée pour indiquer que l'attribut dateSoumission est de type temporel.

Remarques

Gestion des Erreurs : Les erreurs sont encapsulées dans l'attribut error de type GenericError.

Extensibilité : La classe est conçue pour être facilement extensible pour inclure d'autres attributs ou méthodes si nécessaire.

Configuration : La classe utilise des annotations pour le formatage JSON et la gestion des dates, permettant une intégration facile avec les systèmes utilisant ces formats.

Coordination : La classe TransferApiAmplitudeResponse coordonne la gestion des réponses de l'API, assurant une représentation claire et structurée des informations de transfert.
