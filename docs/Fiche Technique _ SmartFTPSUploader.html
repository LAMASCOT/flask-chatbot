SmartFTPSUploader

Introduction

Description : La tache SmartFTPSUploader est conçue pour gérer le téléchargement de fichiers via le protocole FTPS (FTP Secure). Elle permet de télécharger des fichiers depuis un serveur distant, de vérifier s'ils ont déjà été traités, et de les supprimer du serveur après téléchargement si nécessaire.

Description Fonctionnelle

Déclenchement : La méthode treatFile() est appelée pour initier le processus de téléchargement.

Traitement : Télécharge les fichiers depuis un serveur FTPS, vérifie s'ils ont déjà été traités, et les supprime du serveur si configuré.

Résultat : Les fichiers sont téléchargés dans un répertoire local spécifié.

En cas d’erreur : Les erreurs de connexion ou de téléchargement sont consignées dans les journaux.

Opérations Clés

Connexion FTPS : Établit une connexion sécurisée avec le serveur FTPS en utilisant les paramètres de connexion fournis.

Téléchargement de Fichiers : Télécharge les fichiers qui correspondent à une extension spécifique depuis un répertoire distant.

Vérification de Traitement : Vérifie si un fichier a déjà été traité en consultant une base de données.

Suppression de Fichiers : Supprime les fichiers du serveur distant après téléchargement si configuré pour le faire.

Décryptage de Mot de Passe : Décrypte le mot de passe FTPS si celui-ci est stocké de manière cryptée.

Paramètres du Composant

SFTPHOST : Adresse du serveur FTPS, obtenue dynamiquement.

SFTPPORT : Port du serveur FTPS, avec comme valeur par défaut de 22.

SFTPUSER : Nom d'utilisateur pour la connexion FTPS, obtenu dynamiquement.

SFTPPASSWORD : Mot de passe pour la connexion FTPS, qui peut être crypté, obtenu dynamiquement.

REMOTE_FOLDER : Répertoire distant à partir duquel les fichiers doivent être téléchargés , avec comme valeur par défaut /remote/incoming/ (configurable).

REMOTE_FILE_EXTENSION : Extension des fichiers à télécharger, avec comme valeur par défaut *.

LOCAL_FOLDER : Répertoire local où les fichiers téléchargés seront stockés, avec comme valeur par défaut /var/data/sftp_upload/.

DEL_FILE_AFTER_DOWNL : Indicateur pour supprimer les fichiers du serveur après téléchargement, avec comme valeur par défaut false.

Exemples d'Utilisation

Utilisée pour automatiser le téléchargement sécurisé de fichiers depuis un serveur FTPS dans un environnement d'entreprise, en assurant que les fichiers sont correctement téléchargés et traités.

Intégration avec d'autres Systèmes

Intégrée avec des systèmes de gestion de fichiers et des bases de données pour assurer un traitement fiable et efficace des fichiers téléchargés.
