Introduction

Description : La tache Icom3SrgWriter est une extension de FlatFileWriter utilisée pour générer et envoyer des fichiers ICOM3 sous-régionaux vers la BCEAO. Elle traite divers types d'opérations financières telles que les chèques, effets, virements, et prélèvements, en fonction de leur état et type.

Package

Emplacement : sas.action.writers.sica

Description Fonctionnelle

Objectif : Générer et envoyer des fichiers ICOM3 sous-régionaux vers la BCEAO en traitant les opérations financières disponibles.

Traitement :

Vérifie les paramètres de compensation manuelle et d'envoi.

Récupère les opérations financières (chèques, effets, virements, prélèvements) d'une banque.

Prépare les opérations en fonction de leur type et destination.

Met à jour l'état des opérations après traitement.

Méthodes

Icom3SrgWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus de génération et d'envoi des fichiers ICOM3.

execute_BKP() : Méthode de sauvegarde pour l'exécution du processus.

prepareRemiseAnnulation() : Prépare l'annulation de remise pour l'envoi.

annulationTotale(DataBaseJPA db) : Gère l'annulation totale des remises, lots, ou ordres.

preparePrelevements(List<Prelevements> prelevements, Lotcom lotcom1) : Prépare les prélèvements pour l'envoi.

Paramètres Utilisés

CETAOPEALLICOM3 : État des opérations à traiter pour ICOM3.

CETAOPEALLICOM3TRAN : État transitoire des opérations après traitement ICOM3.

COMPENSATION_MANUELLE : Indicateur de compensation manuelle.

DATECOMPENS_SRG : Date de compensation sous-régionale.

VERSION_SICA : Version du système SICA utilisé.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération ou de l'envoi des fichiers sont gérées par des logs et des mises à jour de paramètres.

Extensibilité : La tache peut être étendue pour inclure d'autres types d'opérations ou de traitements spécifiques.

Intégration : Cette tache utilise DataBaseJPA pour interagir avec la base de données et récupérer les opérations à traiter.
