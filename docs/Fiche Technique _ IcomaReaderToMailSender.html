Introduction

Description : La tache IcomaReaderToMailSender est une extension de FlatFileReader qui combine la lecture de fichiers ICOMA et l'envoi de rapports par email. Elle utilise IcomaReader pour traiter les fichiers et IcomaMailSender pour envoyer les rapports générés.

Package

Emplacement : sas.action.writers.sica.mail

Description Fonctionnelle

Objectif : Traiter les fichiers ICOMA et envoyer les rapports correspondants par email.

Traitement :

Utilise IcomaReader pour lire et traiter les fichiers ICOMA.

Utilise IcomaMailSender pour générer et envoyer les rapports par email.

Renomme les fichiers traités pour indiquer qu'ils ont été terminés.

Méthodes

treatFile(File aFile, Repertoires repertoire) : Traite le fichier ICOMA en utilisant IcomaReader, puis envoie le rapport par email en utilisant IcomaMailSender. Renomme le fichier traité pour indiquer qu'il est terminé.

closeFile() : Ferme le fichier après traitement, en appelant la méthode de la classe parente.

Paramètres Utilisés

TEMPLATE_ICOMA_REJET : Chemin du modèle d'email pour les rejets, utilisé par IcomaMailSender.

TEMPLATE_ICOMA_ACCEPTE : Chemin du modèle d'email pour les acceptations, utilisé par IcomaMailSender.

EMAILS_DEST_OPS : Liste des destinataires des emails, utilisé par IcomaMailSender.

SEND_PDF_REPORT : Indicateur pour envoyer ou non le rapport PDF, utilisé par IcomaMailSender.

CONSTANTE_SICA : Utilisé pour déterminer la source dans IcomaMailSender.

Remarques

Gestion des Erreurs : Les erreurs lors du traitement des fichiers ou de l'envoi des emails sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe peut être étendue pour inclure d'autres types de traitement ou de rapports.

Intégration : Cette classe intègre les fonctionnalités de lecture et d'envoi de rapports, simplifiant ainsi le flux de travail pour le traitement des fichiers ICOMA.
