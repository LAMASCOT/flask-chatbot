Introduction

Description : La tache GenOrdersByBankForSFDWriter est une extension de FlatFileWriter utilisée pour générer des fichiers d'ordres pour les Systèmes Financiers Décentralisés (SFD) à destination des banques. Elle coordonne l'exécution de deux autres écrivains de fichiers, MailiBankSFDWriter et LotSFDBankWriter, pour produire les fichiers nécessaires.

Package

Emplacement : sas.action.writers.corporates.sfd.sica

Description Fonctionnelle

Objectif : Générer et coordonner la création de fichiers d'ordres pour les SFD vers les banques.

Traitement :

Initialise et exécute les écrivains de fichiers MailiBankSFDWriter et LotSFDBankWriter.

Met à jour la description du processus en fonction des résultats des écrivains de fichiers.

Attributs

mailiNatBankSFDWriter : MailiBankSFDWriter - Instance de l'écrivain de fichiers pour les ordres de type Maili.

lotSFDBankWriter : LotSFDBankWriter - Instance de l'écrivain de fichiers pour les lots de SFD.

Méthodes

GenOrdersByBankForSFDWriter() : Constructeur par défaut qui initialise les écrivains de fichiers et la description du processus.

execute() : Exécute le processus de génération des fichiers d'ordres en coordonnant les écrivains de fichiers.

Paramètres Utilisés

ParametersMap : Utilisé pour passer des paramètres de configuration aux écrivains de fichiers MailiBankSFDWriter et LotSFDBankWriter.

Remarques

Coordination : La classe GenOrdersByBankForSFDWriter agit comme un coordinateur, orchestrant l'exécution de plusieurs écrivains de fichiers pour assurer la génération complète des fichiers d'ordres pour les SFD.

Extensibilité : La classe est conçue pour être facilement extensible, permettant l'ajout de nouveaux écrivains de fichiers ou de nouvelles fonctionnalités si nécessaire.

Gestion des Erreurs : Les erreurs lors de l'exécution des écrivains de fichiers sont gérées par les classes respectives, et la description du processus est mise à jour en conséquence.
