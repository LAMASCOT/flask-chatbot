Introduction

Description : La tache CompteBulkReader est utilisée pour lire et traiter les fichiers contenant des informations de comptes bancaires, en effectuant des opérations de suppression et d'insertion en masse sur ces comptes dans la base de données.

Package

Emplacement : sas.action.readers.comptes

Description Fonctionnelle

Déclenchement : Activée lors de la détection de nouveaux fichiers de comptes dans le répertoire configuré.

Traitement :

Lit chaque ligne du fichier pour extraire les informations de comptes bancaires.

Supprime tous les comptes existants dans la base de données avant d'insérer les nouveaux comptes extraits du fichier.

Utilise des transactions en lot pour insérer les comptes dans la base de données.

Résultat : Les comptes bancaires sont supprimés et réinsérés dans la base de données selon les instructions du fichier.

En cas d’erreur : Les erreurs de traitement sont consignées dans les journaux pour un suivi ultérieur.

Opérations Clés

Lecture des Fichiers : Utilise LineIterator pour lire chaque ligne du fichier et extraire les informations nécessaires.

Gestion des Comptes :

Supprime tous les comptes existants dans la base de données.

Insère les nouveaux comptes extraits du fichier.

Interaction avec la Base de Données :

Utilise DataBaseJPA pour exécuter des requêtes SQL et gérer les transactions de base de données.

Utilise des transactions en lot pour optimiser l'insertion des comptes.

Gestion des Erreurs : Consigne les erreurs de validation et de traitement dans les journaux.

Paramètres du Composant

Séparateur de Fichier de Compte : Utilise un séparateur de fichier configurable pour analyser les lignes du fichier.

Cette fiche technique inclut les fonctionnalités principales de la classe CompteBulkReader, en mettant l'accent sur le processus de lecture, de gestion des comptes et d'interaction avec la base de données.
