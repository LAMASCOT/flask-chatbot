Introduction

Description : La tache OrdFileWriter est une extension de FlatFileWriter utilisée pour générer et envoyer des fichiers ORD. Ces fichiers contiennent des informations sur les chèques à traiter, formatées selon des spécifications précises.

Package

Emplacement : sas.action.writers.uap

Description Fonctionnelle

Objectif : Générer des fichiers ORD contenant des informations sur les chèques pour un traitement ultérieur.

Traitement :

Formate les informations des chèques en une ligne de texte selon un format prédéfini.

Écrit ces lignes dans un fichier ORD.

Gère la séquence et le nommage des fichiers pour assurer l'unicité et la traçabilité.

Méthodes

OrdFileWriter() : Constructeur par défaut qui initialise la description de l'opération.

OrdFileWriter(Cheques[] cheques35, String sequence) : Constructeur qui initialise les chèques et la séquence pour le fichier ORD.

execute() : Exécute le processus de génération du fichier ORD.

listOfFiles(String path) : Retourne la liste des fichiers dans un répertoire donné, avec des logs pour chaque fichier ou répertoire trouvé.

listFilesForFolder(File folder) : Parcourt récursivement un dossier pour lister tous les fichiers, avec des logs pour chaque fichier trouvé.

getCheques35() : Retourne le tableau de chèques.

setCheques35(Cheques[] cheques35) : Définit le tableau de chèques.

Paramètres Utilisés

ORD_FOLDER : Paramètre indiquant le répertoire où les fichiers ORD doivent être enregistrés.

CODE_BANQUE_SICA3 : Code de la banque utilisé pour déterminer le préfixe du fichier ORD.

Remarques

Gestion des Erreurs : Les erreurs lors de la génération des fichiers sont gérées par des logs et des messages d'erreur.

Extensibilité : La classe peut être étendue pour inclure d'autres types de fichiers ou formats de sortie.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
