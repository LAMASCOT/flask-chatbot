Introduction

Description : La tache ChequeRetourSaisiSitWriter est une extension de FlatFileWriter utilisée pour envoyer les chèques de retour saisis vers le SIB (Système d'Information Bancaire). Elle génère un fichier plat contenant les informations des chèques de retour saisis à envoyer et met à jour leur état dans la base de données.

Package

Emplacement : sas.action.writers.orion

Description Fonctionnelle

Objectif : Envoyer les chèques de retour saisis vers le SIB en générant un fichier plat et mettre à jour leur état dans la base de données.

Traitement :

Met à jour l'état des chèques avec des montants manquants.

Récupère les chèques de retour saisis à envoyer.

Génère un fichier plat avec les informations des chèques de retour saisis.

Met à jour l'état des chèques après envoi.

Méthodes

ChequeRetourSaisiSitWriter() : Constructeur par défaut qui initialise la description de l'envoi.

execute() : Exécute le processus d'envoi des chèques de retour saisis vers le SIB.

Paramètres Utilisés

CETAOPEERR : État des chèques avec erreur de validation.

CETAOPEVAL : État des chèques prêts à être validés.

CETAOPERETMANENVSIB : État des chèques de retour manuels après envoi.

Remarques

Gestion des Erreurs : Les erreurs lors de l'envoi des chèques de retour saisis sont gérées par des logs.

Extensibilité : La tache peut être étendue pour inclure d'autres types de chèques ou de traitements spécifiques.

Intégration : Cette tache utilise DataBase pour interagir avec la base de données et effectuer les mises à jour nécessaires.
