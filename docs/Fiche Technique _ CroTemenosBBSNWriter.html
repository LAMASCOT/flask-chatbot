Introduction

Description : La tache CroTemenosBBSNWriter est une extension de FlatFileWriter utilisée pour gérer l'envoi de lots d'opérations de retour vers le SIB (Système d'Information Bancaire). Elle traite divers types d'opérations financières telles que les prélèvements, chèques, virements, et effets, en les préparant pour l'envoi.

Package

Emplacement : sas.action.writers.temenos

Description Fonctionnelle

Objectif : Préparer et envoyer des lots d'opérations financières de retour vers le SIB.

Traitement :

Récupère les opérations financières depuis la base de données.

Prépare les opérations en fonction de leur type (prélèvements, chèques, virements, effets).

Gère les rejets d'opérations.

Met à jour l'état des opérations après traitement.

Méthodes

CroTemenosBBSNWriter() : Constructeur par défaut qui initialise la description de l'opération.

execute() : Exécute le processus de préparation et d'envoi des lots d'opérations.

preparePrelevements(List<Prelevements> prelevements) : Prépare les prélèvements pour l'envoi.

prepareRejetPrelevements(List<Prelevements> prelevements) : Prépare les rejets de prélèvements.

prepareRejetCheques(List<Cheques> cheques) : Prépare les rejets de chèques.

prepareRejetEffets(List<Effets> effets) : Prépare les rejets d'effets.

prepareCheques(List<Cheques> cheques) : Prépare les chèques pour l'envoi.

prepareEffets(List<Effets> effets) : Prépare les effets pour l'envoi.

fetchCompte(String numero) : Récupère les informations de compte à partir du numéro de compte.

Paramètres Utilisés

CETAOPERETREC : Paramètre indiquant l'état des opérations de retour reçues.

CETAOPERET : Paramètre indiquant l'état des opérations de retour.

CETAOPEREJRET : Paramètre indiquant l'état des rejets d'opérations de retour.

Remarques

Gestion des Erreurs : Les erreurs lors de la préparation des opérations sont gérées par des logs.

Extensibilité : La classe peut être étendue pour inclure d'autres types d'opérations ou pour intégrer d'autres systèmes de gestion de fichiers.

Utilisation de la Bibliothèque Apache Commons Lang : La classe utilise StringUtils de la bibliothèque Apache Commons Lang pour la manipulation des chaînes de caractères.
